Trait AsPrimitive

pub trait AsPrimitive<T> {
    // Required methods
    pub fn as_(self) -> T;
}

A generic interface for casting between primitive types, equivalent of using the as keyword between values.

Example

let x: Field = 1234567890;
let y: u8 = x as u8;
let z: u8 = x.as_();
assert_eq(y, z);

Required methods

pub fn as_(self) -> T

The equivalent of doing self as T.
