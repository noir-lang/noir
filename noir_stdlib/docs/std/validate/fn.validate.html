Function validate

pub comptime fn validate(s: TypeDefinition, method: FunctionDefinition)

Attribute backing #[validate(Type::method)]. It does not generate any code itself — the #[derive(Validate)] handler reads these attributes and splices the calls — but it checks that each named method has the shape the derived validate expects to call: a single self argument and no return value. Reporting that here points the error at the attribute rather than at the generated impl.
