1// Re-usable methods that backends can use to implement their PWG
2
3//! This module contains methods to implement the partial witness generation (PWG) of an ACIR program.
4//! The goal of ACIR execution is to compute the values of all the ACIR witnesses, or an error if it could not compute them all.
5//! A proving system will then be able to use the ACIR circuit and the values of the ACIR witnesses to generate a proof of this execution.
6//! The ACIR opcodes are not modified by the execution.
7//! Witness generation means getting valid values for the witnesses used by the ACIR opcodes of the program.
8//! They are called *partial* witness because a proving system may create additional witnesses on its own for
9//! generating the proof (and a corresponding low-level circuit). The PWG generates values for all the witnesses
10//! of the ACIR program, or returns an error if it cannot do it.
11//!
12//! Implementation details & examples:
13//! It starts by instantiating an ACVM (ACIR Virtual Machine), which executes the given ACIR opcodes in the `solve()` function.
14//!
15//! Parameters: When instantiating the ACVM, it needs to be provided with:
16//!  - a `backend` implementing the `BlackBoxFunctionSolver` trait. Different implementation can be used depending on the EC used by the underlying proving system.
17//!  - `opcodes`: the ACIR opcodes of the program to solve.
18//!  - `initial_witness`: a mapping of initial witness values representing the inputs of the program. The ACVM will update this map as it solves the opcodes.
19//!  - `unconstrained_functions`: the Brillig bytecode of the unconstrained functions used by the program.
20//!  - `assertion_payloads`: additional information used to provide feedback to the user when an assertion fails.
21//!
22//! Returns: [`ACVMStatus`]
23//!
24//! Each opcode is solved independently. In general we require its inputs to be already known, i.e previously solved,
25//! and the output is simply computed from the inputs, and then the output becomes 'known' for the subsequent opcodes.
26//!
27//! See [`acir::circuit::Opcode`] for more details.
28//!
29//! Example:
30// Compiled ACIR for main (non-transformed):
31// func 0
32// private parameters: [w0, w1, w2, w3, w4]
33// public parameters: []
34// return values: [w9]
35// BLACKBOX::RANGE input: w0, bits: 32
36// BLACKBOX::RANGE input: w1, bits: 32
37// BLACKBOX::RANGE input: w2, bits: 32
38// BLACKBOX::RANGE input: w3, bits: 32
39// BLACKBOX::RANGE input: w4, bits: 32
40// ASSERT w0 - w1 - w6 = 0
41// BRILLIG CALL func: 0, predicate: 1, inputs: [w6], outputs: [w7]
42// ASSERT w6*w7 + w8 - 1 = 0
43// ASSERT w6*w8 = 0
44// ASSERT w1*w8 = 0
45// ASSERT w0 - w2 - w9 = 0
46//!
47//! This ACIR program defines the 'main' function and indicates it is 'non-transformed'.
48//! Indeed, some ACIR pass can transform the ACIR program in order to apply optimizations,
49//! or to make it compatible with a specific proving system.
50//! However, ACIR execution is expected to work on any ACIR program (transformed or not).
51//! Then we see the parameters of the program as public and private inputs.
52//! The `initial_witness` needs to contain values for these parameters before execution, else
53//! the execution will fail.
54//! The first ACIR opcodes are RANGE opcodes which ensure the inputs have the expected range (as specified in the Noir source code).
55//! Solving this black-box simply means to validate that the values (from `initial_witness`) are indeed 32 bits for w0, w1, w2, w3, w4
56//! If `initial_witness` does not have values for w0, w1, w2, w3, w4, or if the values are over 32 bits, the execution will fail.
57//! The next opcode is an AssertZero opcode: ASSERT w0 - w1 - w6 = 0, which indicates that `w0 - w1 - w6` should be equal to 0.
58//! Since we know the values of `w0, w1` from `initial_witness`, we can compute `w6 = w0 + w1` so that the AssertZero is satisfied.
59//! Solving AssertZero means computing the unknown witness and adding the result to `initial_witness`, which now contains the value for `w6`.
60//! The next opcode is a Brillig Call where input is `w6` and output is `w7`. From the function id of the opcode, the solver will retrieve the
61//! corresponding Brillig bytecode and instantiate a Brillig VM with the value of the input. This value was just computed before.
62//! Executing the Brillig VM on this input will give us the output which is the value for `w7`, that we add to `initial_witness`.
63//! The next opcode is again an AssertZero: `w6 * w7 + w8 - 1 = 0`, which computes the value of `w8`.
64//! The two next opcodes are AssertZero without any unknown witnesses: `w6 * w8 = 0` and `w1 * w8 = 0`
65//! Solving such opcodes means that we compute `w6 * w8 ` and `w1 * w8` using the known values, and check that they evaluate to 0.
66//! If not, we would return an error.
67//! Finally, the last AssertZero computes `w9` which is the last witness. All of the witnesses have now been computed; execution is complete.
68

102#[derive(Debug, Clone, PartialEq)]
103pub enum ACVMStatus<F> {
104    /// All witnesses have been computed and all opcodes have been successfully resolved. Execution is complete.
105    Solved,
106
107    /// The ACVM is processing the circuit, i.e solving the opcodes. This status is used to resume execution after it has been paused.
108    InProgress,
109
110    /// The ACVM has encountered an irrecoverable error while executing the circuit and can not progress.
111    /// Most commonly this will be due to an unsatisfied constraint due to invalid inputs to the circuit.
112    Failure(OpcodeResolutionError<F>),
113
114    /// The ACVM has encountered a request for a Brillig [foreign call][brillig_vm::brillig::Opcode::ForeignCall]
115    /// to retrieve information from outside of the ACVM. The result of the foreign call must be passed back
116    /// to the ACVM using [`ACVM::resolve_pending_foreign_call`].
117    ///
118    /// Once this is done, the ACVM can be restarted to solve the remaining opcodes.
119    RequiresForeignCall(ForeignCallWaitInfo<F>),
120
121    /// The ACVM has encountered a request for an ACIR [call][acir::circuit::Opcode]
122    /// to execute a separate ACVM instance. The result of the ACIR call must be passed back to the ACVM.
123    ///
124    /// Once this is done, the ACVM can be restarted to solve the remaining opcodes.
125    RequiresAcirCall(AcirCallWaitInfo<F>),
126}
127
128impl<F> std::fmt::Display for ACVMStatus<F> {
129    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
130        match self {
131            ACVMStatus::Solved => write!(f, "Solved"),
132            ACVMStatus::InProgress => write!(f, "In progress"),
133            ACVMStatus::Failure(_) => write!(f, "Execution failure"),
134            ACVMStatus::RequiresForeignCall(_) => write!(f, "Waiting on foreign call"),
135            ACVMStatus::RequiresAcirCall(_) => write!(f, "Waiting on acir call"),
136        }
137    }
138}

205#[derive(Clone, PartialEq, Eq, Debug, Error)]
206pub enum OpcodeResolutionError<F> {
207    #[error("Cannot solve opcode: {0}")]
208    OpcodeNotSolvable(#[from] OpcodeNotSolvable<F>),
209    #[error("Cannot satisfy constraint")]
210    UnsatisfiedConstrain {
211        opcode_location: ErrorLocation,
212        payload: Option<ResolvedAssertionPayload<F>>,
213    },
214    #[error("Index out of bounds, array has size {array_size:?}, but index was {index:?}")]
215    IndexOutOfBounds { opcode_location: ErrorLocation, index: F, array_size: u32 },
216    #[error("Cannot solve opcode: {invalid_input_bit_size}")]
217    InvalidInputBitSize {
218        opcode_location: ErrorLocation,
219        invalid_input_bit_size: InvalidInputBitSize,
220    },
221    #[error("Failed to solve blackbox function: {0}, reason: {1}")]
222    BlackBoxFunctionFailed(BlackBoxFunc, String),
223    #[error("Failed to solve brillig function")]
224    BrilligFunctionFailed {
225        function_id: BrilligFunctionId,
226        call_stack: Vec<OpcodeLocation>,
227        payload: Option<ResolvedAssertionPayload<F>>,
228    },
229    #[error("Attempted to call `main` with a `Call` opcode")]
230    AcirMainCallAttempted { opcode_location: ErrorLocation },
231    #[error(
232        "{results_size:?} result values were provided for {outputs_size:?} call output witnesses, most likely due to bad ACIR codegen"
233    )]
234    AcirCallOutputsMismatch { opcode_location: ErrorLocation, results_size: u32, outputs_size: u32 },
235    #[error("(--pedantic): Predicates are expected to be 0 or 1, but found: {pred_value}")]
236    PredicateLargerThanOne { opcode_location: ErrorLocation, pred_value: F },
237    #[error("(--pedantic): Memory operations are expected to be 0 or 1, but found: {operation}")]
238    MemoryOperationLargerThanOne { opcode_location: ErrorLocation, operation: F },
239}
240
241impl<F> From<BlackBoxResolutionError> for OpcodeResolutionError<F> {
242    fn from(value: BlackBoxResolutionError) -> Self {
243        match value {
244            BlackBoxResolutionError::Failed(func, reason) => {
245                OpcodeResolutionError::BlackBoxFunctionFailed(func, reason)
246            }
247            BlackBoxResolutionError::AssertFailed(error) => {
248                OpcodeResolutionError::UnsatisfiedConstrain {
249                    opcode_location: ErrorLocation::Unresolved,
250                    payload: Some(ResolvedAssertionPayload::String(error)),
251                }
252            }
253        }
254    }
255}

257impl<F> From<InvalidInputBitSize> for OpcodeResolutionError<F> {

259        Self::InvalidInputBitSize {
260            opcode_location: ErrorLocation::Unresolved,
261            invalid_input_bit_size,
262        }
263    }

430    /// Resolves a foreign call's [result][brillig_vm::brillig::ForeignCallResult] using a result calculated outside of the ACVM.
431    ///
432    /// The ACVM can then be restarted to solve the remaining Brillig VM process as well as the remaining ACIR opcodes.
433    pub fn resolve_pending_foreign_call(&mut self, foreign_call_result: ForeignCallResult<F>) {
434        if !matches!(self.status, ACVMStatus::RequiresForeignCall(_)) {
435            panic!("ACVM is not expecting a foreign call response as no call was made");
436        }
437
438        let brillig_solver = self.brillig_solver.as_mut().expect("No active Brillig solver");
439        brillig_solver.resolve_pending_foreign_call(foreign_call_result);
440
441        // Now that the foreign call has been resolved then we can resume execution.
442        self.status(ACVMStatus::InProgress);
443    }

445    /// Sets the status of the VM to `RequiresAcirCall`
446    /// Indicating that the VM is now waiting for an ACIR call to be resolved
447    fn wait_for_acir_call(&mut self, acir_call: AcirCallWaitInfo<F>) -> ACVMStatus<F> {
448        self.status(ACVMStatus::RequiresAcirCall(acir_call))
449    }

451    /// Resolves an ACIR call's result (simply a list of fields) using a result calculated by a separate ACVM instance.
452    ///
453    /// The current ACVM instance can then be restarted to solve the remaining ACIR opcodes.
454    pub fn resolve_pending_acir_call(&mut self, call_result: Vec<F>) {
455        if !matches!(self.status, ACVMStatus::RequiresAcirCall(_)) {
456            panic!("ACVM is not expecting an ACIR call response as no call was made");
457        }
458
459        if self.acir_call_counter < self.acir_call_results.len() {
460            panic!("No unresolved ACIR calls");
461        }
462        self.acir_call_results.push(call_result);
463
464        // Now that the ACIR call has been resolved then we can resume execution.
465        self.status(ACVMStatus::InProgress);
466    }
467
468    /// Executes the ACVM's circuit until execution halts.
469    ///
470    /// Execution can halt due to three reasons:
471    /// 1. All opcodes have been executed successfully.
472    /// 2. The circuit has been found to be unsatisfiable.
473    /// 2. A Brillig [foreign call][`ForeignCallWaitInfo`] has been encountered and must be resolved.
474    pub fn solve(&mut self) -> ACVMStatus<F> {
475        while self.status == ACVMStatus::InProgress {
476            self.solve_opcode();
477        }
478        self.status.clone()
479    }
480
481    fn current_opcode(&self) -> &'a Opcode<F> {
482        &self.opcodes[self.instruction_pointer]
483    }
484
485    /// Executes a single opcode using the dedicated solver.
486    ///
487    /// Foreign or ACIR Calls are deferred to the caller, which will
488    /// either instantiate a new ACVM to execute the called ACIR function
489    /// or a custom implementation to execute the foreign call.
490    /// Then it will resume execution of the current ACVM with the results of the call.
491    pub fn solve_opcode(&mut self) -> ACVMStatus<F> {
492        let resolution = match self.current_opcode() {
493            Opcode::AssertZero(expr) => ExpressionSolver::solve(&mut self.witness_map, expr),
494            Opcode::BlackBoxFuncCall(bb_func) => {
495                blackbox::solve(self.backend, &mut self.witness_map, bb_func)
496            }
497            Opcode::MemoryInit { block_id, init, .. } => {
498                self.solve_memory_init_opcode(*block_id, init)
499            }
500            Opcode::MemoryOp { block_id, op } => {
501                let solver = self
502                    .block_solvers
503                    .get_mut(block_id)
504                    .expect("Memory block should have been initialized before use");
505                solver.solve_memory_op(op, &mut self.witness_map)
506            }
507            Opcode::BrilligCall { id, inputs, outputs, predicate } => {
508                match self.solve_brillig_call_opcode(id, inputs, outputs, predicate) {
509                    Ok(Some(foreign_call)) => return self.wait_for_foreign_call(foreign_call),
510                    res => res.map(|_| ()),
511                }
512            }
513            Opcode::Call { id, inputs, outputs, predicate } => {
514                match self.solve_call_opcode(id, inputs, outputs, predicate) {
515                    Ok(Some(input_values)) => return self.wait_for_acir_call(input_values),
516                    res => res.map(|_| ()),
517                }
518            }
519        };
520        self.handle_opcode_resolution(resolution)
521    }
522
523    /// Returns the status of the ACVM
524    /// If the status is an error, it converts the error into [OpcodeResolutionError]
525    fn handle_opcode_resolution(
526        &mut self,
527        resolution: Result<(), OpcodeResolutionError<F>>,
528    ) -> ACVMStatus<F> {
529        match resolution {
530            Ok(()) => {
531                self.instruction_pointer += 1;
532                if self.instruction_pointer == self.opcodes.len() {
533                    self.status(ACVMStatus::Solved)
534                } else {
535                    self.status(ACVMStatus::InProgress)
536                }
537            }
538            Err(mut error) => {
539                match &mut error {
540                    // If we have an index out of bounds, unsatisfied constraint, or an invalid input bit size,
541                    // the opcode label will be unresolved because the solvers do not have knowledge of this information.
542                    // We resolve, by setting this to the corresponding opcode that we just attempted to solve.
543                    OpcodeResolutionError::IndexOutOfBounds {
544                        opcode_location: opcode_index,
545                        ..
546                    } => {
547                        *opcode_index = ErrorLocation::Resolved(OpcodeLocation::Acir(
548                            self.instruction_pointer(),
549                        ));
550                    }
551                    OpcodeResolutionError::UnsatisfiedConstrain {
552                        opcode_location: opcode_index,
553                        payload: assertion_payload,
554                    } => {
555                        let location = OpcodeLocation::Acir(self.instruction_pointer());
556                        *opcode_index = ErrorLocation::Resolved(location);
557                        *assertion_payload = self.extract_assertion_payload(location);
558                    }
559                    OpcodeResolutionError::InvalidInputBitSize {
560                        opcode_location: opcode_index,
561                        ..
562                    } => {
563                        let location = OpcodeLocation::Acir(self.instruction_pointer());
564                        *opcode_index = ErrorLocation::Resolved(location);
565                    }
566                    // All other errors are thrown normally.
567                    _ => (),
568                }
569                self.fail(error)
570            }
571        }
572    }
573
574    fn extract_assertion_payload(
575        &self,
576        location: OpcodeLocation,
577    ) -> Option<ResolvedAssertionPayload<F>> {
578        let (_, assertion_descriptor) =
579            self.assertion_payloads.iter().find(|(loc, _)| location == *loc)?;
580        let mut fields = Vec::new();
581        for expr in &assertion_descriptor.payload {
582            match expr {
583                ExpressionOrMemory::Expression(expr) => {
584                    let value = get_value(expr, &self.witness_map).ok()?;
585                    fields.push(value);
586                }
587                ExpressionOrMemory::Memory(block_id) => {
588                    let memory_block = self.block_solvers.get(block_id)?;
589                    fields.extend(&memory_block.block_value);
590                }
591            }
592        }
593        let error_selector = ErrorSelector::new(assertion_descriptor.error_selector);
594
595        Some(ResolvedAssertionPayload::Raw(RawAssertionPayload {
596            selector: error_selector,
597            data: fields,
598        }))
599    }

601    /// Initializes a memory block with values loaded from the witness map.
602    ///
603    /// Fails if the block has already been initialized.
604    fn solve_memory_init_opcode(
605        &mut self,
606        block_id: BlockId,
607        init: &[Witness],
608    ) -> Result<(), OpcodeResolutionError<F>> {
609        let solver = MemoryOpSolver::new(init, &self.witness_map)?;
610        if self.block_solvers.insert(block_id, solver).is_some() {
611            return Err(OpcodeResolutionError::UnsatisfiedConstrain {
612                opcode_location: ErrorLocation::Unresolved,
613                payload: None,
614            });
615        }
616        Ok(())
617    }

619    /// Solves a Brillig Call opcode, which represents a call to an unconstrained function.
620    /// It first handles the predicate and returns zero values if the predicate is false.
621    /// Then it executes (or resumes execution) the Brillig function using a Brillig VM.
622    fn solve_brillig_call_opcode(
623        &mut self,
624        id: &BrilligFunctionId,
625        inputs: &'a [BrilligInputs<F>],
626        outputs: &[BrilligOutputs],
627        predicate: &Expression<F>,
628    ) -> Result<Option<ForeignCallWaitInfo<F>>, OpcodeResolutionError<F>> {
629        let opcode_location =
630            ErrorLocation::Resolved(OpcodeLocation::Acir(self.instruction_pointer()));
631        if id.as_usize() >= self.unconstrained_functions.len() {
632            return Err(OpcodeResolutionError::BrilligFunctionFailed {
633                function_id: *id,
634                call_stack: vec![OpcodeLocation::Acir(self.instruction_pointer())],
635                payload: None,
636            });
637        }
638        if is_predicate_false(&self.witness_map, predicate, &opcode_location)? {
639            return BrilligSolver::<F, B>::zero_out_brillig_outputs(&mut self.witness_map, outputs)
640                .map(|_| None);
641        }
642
643        // If we're resuming execution after resolving a foreign call then
644        // there will be a cached `BrilligSolver` to avoid recomputation.
645        let mut solver: BrilligSolver<'_, F, B> = match self.brillig_solver.take() {
646            Some(solver) => solver,
647            None => BrilligSolver::new_call(
648                &self.witness_map,
649                &self.block_solvers,
650                inputs,
651                &self.unconstrained_functions[id.as_usize()].bytecode,
652                self.backend,
653                self.instruction_pointer,
654                *id,
655                self.profiling_active,
656                self.brillig_branch_to_feature_map,
657            )?,
658        };
659
660        // If we're fuzzing, we need to get the fuzzing trace on an error
661        let result = solver.solve().inspect_err(|_| {
662            if self.brillig_fuzzing_active {
663                self.brillig_fuzzing_trace = Some(solver.get_fuzzing_trace());
664            }
665        })?;
666
667        match result {
668            BrilligSolverStatus::ForeignCallWait(foreign_call) => {
669                // Cache the current state of the solver
670                self.brillig_solver = Some(solver);
671                Ok(Some(foreign_call))
672            }
673            BrilligSolverStatus::InProgress => {
674                unreachable!("Brillig solver still in progress")
675            }
676            BrilligSolverStatus::Finished => {
677                if self.brillig_fuzzing_active {
678                    self.brillig_fuzzing_trace = Some(solver.get_fuzzing_trace());
679                }
680                // Write execution outputs
681                if self.profiling_active {
682                    let profiling_info =
683                        solver.finalize_with_profiling(&mut self.witness_map, outputs)?;
684                    profiling_info.into_iter().for_each(|sample| {
685                        let mapped =
686                            sample.call_stack.into_iter().map(|loc| OpcodeLocation::Brillig {
687                                acir_index: self.instruction_pointer,
688                                brillig_index: loc,
689                            });
690                        self.profiling_samples.push(ProfilingSample {
691                            call_stack: std::iter::once(OpcodeLocation::Acir(
692                                self.instruction_pointer,
693                            ))
694                            .chain(mapped)
695                            .collect(),
696                            brillig_function_id: Some(*id),
697                        });
698                    });
699                } else {
700                    solver.finalize(&mut self.witness_map, outputs)?;
701                }
702
703                Ok(None)
704            }
705        }
706    }

708    // This function is used by the debugger
709    pub fn step_into_brillig(&mut self) -> StepResult<'a, F, B> {
710        let Opcode::BrilligCall { id, inputs, outputs, predicate } = self.current_opcode() else {
711            return StepResult::Status(self.solve_opcode());
712        };
713
714        let opcode_location =
715            ErrorLocation::Resolved(OpcodeLocation::Acir(self.instruction_pointer()));
716        if id.as_usize() >= self.unconstrained_functions.len() {
717            return StepResult::Status(self.handle_opcode_resolution(Err(
718                OpcodeResolutionError::BrilligFunctionFailed {
719                    function_id: *id,
720                    call_stack: vec![OpcodeLocation::Acir(self.instruction_pointer())],
721                    payload: None,
722                },
723            )));
724        }
725        let witness = &mut self.witness_map;
726        let should_skip = match is_predicate_false(witness, predicate, &opcode_location) {
727            Ok(result) => result,
728            Err(err) => return StepResult::Status(self.handle_opcode_resolution(Err(err))),
729        };
730        if should_skip {
731            let resolution = BrilligSolver::<F, B>::zero_out_brillig_outputs(witness, outputs);
732            return StepResult::Status(self.handle_opcode_resolution(resolution));
733        }
734
735        let solver = BrilligSolver::new_call(
736            witness,
737            &self.block_solvers,
738            inputs,
739            &self.unconstrained_functions[id.as_usize()].bytecode,
740            self.backend,
741            self.instruction_pointer,
742            *id,
743            self.profiling_active,
744            self.brillig_branch_to_feature_map,
745        );
746        match solver {
747            Ok(solver) => StepResult::IntoBrillig(solver),
748            Err(..) => StepResult::Status(self.handle_opcode_resolution(solver.map(|_| ()))),
749        }
750    }
751
752    // This function is used by the debugger
753    pub fn finish_brillig_with_solver(&mut self, solver: BrilligSolver<'a, F, B>) -> ACVMStatus<F> {
754        if !matches!(self.current_opcode(), Opcode::BrilligCall { .. }) {
755            unreachable!("Not executing a Brillig/BrilligCall opcode");
756        }
757        self.brillig_solver = Some(solver);
758        self.solve_opcode()
759    }

761    /// Defer execution of the ACIR call opcode to the caller, or finalize the execution.
762    /// 1. It first handles the predicate and return zero values if the predicate is false.
763    /// 2. If the results of the execution are not available, it issues a 'AcirCallWaitInfo'
764    ///    to notify the caller that it (the caller) needs to execute the ACIR function.
765    /// 3. If the results are available, it updates the witness map and indicates that the opcode is solved.
766    pub fn solve_call_opcode(
767        &mut self,
768        id: &AcirFunctionId,
769        inputs: &[Witness],
770        outputs: &[Witness],
771        predicate: &Expression<F>,
772    ) -> Result<Option<AcirCallWaitInfo<F>>, OpcodeResolutionError<F>> {
773        let opcode_location =
774            ErrorLocation::Resolved(OpcodeLocation::Acir(self.instruction_pointer()));
775        if *id == AcirFunctionId(0) {
776            return Err(OpcodeResolutionError::AcirMainCallAttempted { opcode_location });
777        }
778
779        if is_predicate_false(&self.witness_map, predicate, &opcode_location)? {
780            // Zero out the outputs if we have a false predicate
781            for output in outputs {
782                insert_value(output, F::zero(), &mut self.witness_map)?;
783            }
784            return Ok(None);
785        }
786
787        if self.acir_call_counter >= self.acir_call_results.len() {
788            let mut initial_witness = WitnessMap::default();
789            for (i, input_witness) in inputs.iter().enumerate() {
790                let input_value = *witness_to_value(&self.witness_map, *input_witness)?;
791                initial_witness.insert(Witness(i as u32), input_value);
792            }
793            return Ok(Some(AcirCallWaitInfo { id: *id, initial_witness }));
794        }
795
796        let result_values = &self.acir_call_results[self.acir_call_counter];
797        if outputs.len() != result_values.len() {
798            return Err(OpcodeResolutionError::AcirCallOutputsMismatch {
799                opcode_location,
800                results_size: result_values.len() as u32,
801                outputs_size: outputs.len() as u32,
802            });
803        }
804
805        for (output_witness, result_value) in outputs.iter().zip_eq(result_values) {
806            insert_value(output_witness, *result_value, &mut self.witness_map)?;
807        }
808
809        self.acir_call_counter += 1;
810        Ok(None)
811    }
812}

814// Returns the concrete value for a particular witness
815// If the witness has no assignment, then
816// an error is returned
817pub fn witness_to_value<F>(
818    initial_witness: &WitnessMap<F>,
819    witness: Witness,
820) -> Result<&F, OpcodeResolutionError<F>> {
821    match initial_witness.get(&witness) {
822        Some(value) => Ok(value),
823        None => Err(OpcodeNotSolvable::MissingAssignment(witness.0).into()),
824    }
825}

827pub fn input_to_value<F: AcirField>(
828    initial_witness: &WitnessMap<F>,
829    input: FunctionInput<F>,
830) -> Result<F, OpcodeResolutionError<F>> {
831    match input {
832        FunctionInput::Witness(witness) => {
833            let initial_value = *witness_to_value(initial_witness, witness)?;
834            Ok(initial_value)
835        }
836        FunctionInput::Constant(value) => Ok(value),
837    }
838}

840pub fn check_bit_size<F: AcirField>(
841    value: F,
842    num_bits: u32,
843) -> Result<(), OpcodeResolutionError<F>> {
844    if value.num_bits() <= num_bits {
845        Ok(())
846    } else {
847        let value_num_bits = value.num_bits();
848        let value = value.to_string();
849        Err(OpcodeResolutionError::InvalidInputBitSize {
850            opcode_location: ErrorLocation::Unresolved,
851            invalid_input_bit_size: InvalidInputBitSize {
852                value,
853                value_num_bits,
854                max_bits: num_bits,
855            },
856        })
857    }
858}

860/// Returns the concrete value for a particular expression
861/// If the value cannot be computed, it returns an 'OpcodeNotSolvable' error.
862pub fn get_value<F: AcirField>(
863    expr: &Expression<F>,
864    initial_witness: &WitnessMap<F>,
865) -> Result<F, OpcodeResolutionError<F>> {
866    if let Some(&c) = expr.to_const() {
867        return Ok(c);
868    }
869    let expr = ExpressionSolver::evaluate(expr, initial_witness);
870    match expr.to_const() {
871        Some(value) => Ok(*value),
872        None => Err(OpcodeResolutionError::OpcodeNotSolvable(
873            OpcodeNotSolvable::MissingAssignment(any_witness_from_expression(&expr).unwrap().0),
874        )),
875    }
876}

878/// Inserts `value` into the initial witness map under the index `witness`.
879///
880/// Returns an error if there was already a value in the map
881/// which does not match the value that one is about to insert
882pub fn insert_value<F: AcirField>(
883    witness: &Witness,
884    value_to_insert: F,
885    initial_witness: &mut WitnessMap<F>,
886) -> Result<(), OpcodeResolutionError<F>> {
887    use std::collections::btree_map::Entry;
888    match initial_witness.entry(*witness) {
889        Entry::Vacant(e) => {
890            e.insert(value_to_insert);
891            Ok(())
892        }
893        Entry::Occupied(e) => {
894            if *e.get() != value_to_insert {
895                Err(OpcodeResolutionError::UnsatisfiedConstrain {
896                    opcode_location: ErrorLocation::Unresolved,
897                    payload: None,
898                })
899            } else {
900                Ok(())
901            }
902        }
903    }
904}

906// Returns one witness belonging to an expression, in no relevant order
907// Returns None if the expression is const
908// The function is used during partial witness generation to report unsolved witness
909fn any_witness_from_expression<F>(expr: &Expression<F>) -> Option<Witness> {
910    if expr.linear_combinations.is_empty() {
911        if expr.mul_terms.is_empty() { None } else { Some(expr.mul_terms[0].1) }
912    } else {
913        Some(expr.linear_combinations[0].1)
914    }
915}

917/// Returns `Ok(true)` if the predicate is zero
918/// A predicate is used to indicate whether we should skip a certain operation.
919/// If we have a zero predicate it means the operation should be skipped.
920pub(crate) fn is_predicate_false<F: AcirField>(
921    witness: &WitnessMap<F>,
922    predicate: &Expression<F>,
923    opcode_location: &ErrorLocation,
924) -> Result<bool, OpcodeResolutionError<F>> {
925    let pred_value = get_value(predicate, witness)?;
926    let predicate_is_false = pred_value.is_zero();
927
928    // We expect that the predicate should resolve to either 0 or 1.
929    if !predicate_is_false && !pred_value.is_one() {
930        let opcode_location = *opcode_location;
931        return Err(OpcodeResolutionError::PredicateLargerThanOne { opcode_location, pred_value });
932    }
933
934    Ok(predicate_is_false)
935}
