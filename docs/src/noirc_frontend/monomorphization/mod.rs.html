1//! Coming after elaboration, monomorphization is the last pass in Noir's frontend.
2//! It accepts the type checked HIR as input and produces a monomorphized AST as output.
3//! This file implements the pass itself, while the AST is defined in the ast module.
4//!
5//! Unlike the HIR, which is stored within the NodeInterner, the monomorphized AST is
6//! self-contained and does not need an external context struct. As a result, the NodeInterner
7//! can be safely discarded after monomorphization.
8//!
9//! The entry point to this pass is the `monomorphize` function which, starting from a given
10//! function, will monomorphize the entire reachable program.
11//!
12//! The monomorphized AST (mAST) has a few notable differences from the HIR:
13//! - It is self-contained without the need for an external context like the NodeInterner.
14//! - All generics are gone, they are specialized away by creating a new copy of each function
15//!   for each combination of generic arguments it is used with.
16//! - All local lambdas are gone and closure environments are explicit. Closures are converted
17//!   into normal tuples of (function, environment) and when the function is called, will forward
18//!   the environment argument to the function as well. All functions are global and local lambdas
19//!   no longer exist.
20//! - Most patterns are simplified away. For example, a tuple pattern in a function parameter:
21//!   `fn foo((a, b): (i32, i32))` will be translated into separate parameters:
22//!   `fn foo(a: i32, b: i32)`. Note that this transformation relies on our SSA construction
23//!   pass to perform a similar translation for the callsites of functions. The monomorphized
24//!   output alone does not change callsites, so it will produce mAST like the following:
25//!   ```noir
26//!   fn main$f0() -> () {
27//!       let tuple$l0 = (0, 1);
28//!       foo$f1(tuple$l0);  // single tuple argument
29//!   }
30//!
31//!   fn foo$f1(a$l1: i32, b$l2: i32) -> () {}  // two separate parameters
32//!   ```
33//! - Functions are represented as a pair of `(constrained, unconstrained)` versions of the same
34//!   function. The variant to use is selected by `Monomorphizer::extract_function` to match the
35//!   runtime when the function is later called.
36//!
37//! At the end of monomorphization, a couple sub-passes are performed:
38//! - [ownership](crate::ownership): infers when values should be cloned or moved for unconstrained code.
39//!   This is only relevant for arrays in unconstrained code which are implemented with copy on
40//!   write semantics. An [ast::Expression::Clone] corresponds to an increment of the reference-count on
41//!   a particular array rather than a deep clone. The deep clone itself will be performed by the
42//!   Brillig runtime when mutating an array with a reference count greater than one.
43//! - [proxies]: wraps oracle functions in unconstrained function wrappers automatically.
44//!   This is required in some corner cases when oracles are used as first-class functions.
45//!
46//! Compared to monomorphization passes in other compilers, Noir's is a bit odd in that it may
47//! still fail with an error message. An example of an error caught at this step would be
48//! an unconstrained lambda being passed into and called in constrained code. This is possible
49//! when a normal lambda is compiled in an unconstrained context and uses types, such as references,
50//! which shouldn't leave the current context.

103/// The context struct for the monomorphization pass.
104///
105/// This struct holds the FIFO queue of functions to monomorphize, which is added to
106/// whenever a new (function, type) combination is encountered.
107pub struct Monomorphizer<'interner> {
108    /// Functions are keyed by their unique ID, whether they're unconstrained, their expected type,
109    /// and any generics they have so that we can monomorphize a new version of the function for each type.
110    ///
111    /// Keying by any turbofish generics that are specified is necessary for a case where we may have a
112    /// trait generic that can be instantiated outside of a function parameter or return value.
113    functions: Functions,
114
115    /// Unlike functions, locals are only keyed by their unique ID because they are never
116    /// duplicated during monomorphization. Doing so would allow them to be used polymorphically
117    /// but would also cause them to be re-evaluated which is a performance trap that would
118    /// confuse users.
119    locals: HashMap<node_interner::DefinitionId, LocalId>,
120
121    /// Globals are keyed by their unique ID and their type, which should create a single global
122    /// instance per generic combination.
123    ///
124    /// The alternative would be to use default types for unused generic parameters,
125    /// but that could fail during SSA validation.
126    globals: HashMap<node_interner::GlobalId, HashMap<HirType, GlobalId>>,
127
128    finished_globals: HashMap<GlobalId, (String, ast::Type, ast::Expression)>,
129
130    /// Queue of functions to monomorphize next each item in the queue is a tuple of:
131    /// (old_id, new_monomorphized_id, any type bindings to apply, the trait method if old_id is from a trait impl, is_unconstrained, location)
132    queue: VecDeque<(
133        node_interner::FuncId,
134        FuncId,
135        TypeBindings,
136        Option<TraitItemId>,
137        bool,
138        Location,
139    )>,
140
141    /// When a function finishes being monomorphized, the monomorphized [ast::Function] is
142    /// stored here along with its [FuncId].
143    finished_functions: BTreeMap<FuncId, Function>,
144
145    /// Used to reference existing definitions in the HIR.
146    interner: &'interner mut NodeInterner,
147
148    lambda_envs_stack: Vec<LambdaContext>,
149
150    next_local_id: u32,
151    next_global_id: u32,
152    next_function_id: u32,
153    next_ident_id: u32,
154
155    /// Location of the last statement in the `main` function, determined during compilation.
156    return_location: Option<Location>,
157
158    debug_type_tracker: DebugTypeTracker,
159
160    /// Indicate that we are currently monomorphizing an unconstrained function, which causes
161    /// constrained function called from this context to be monomorphized as unconstrained too.
162    in_unconstrained_function: bool,
163
164    /// Set to true to force every function to be unconstrained.
165    /// Note that this also changes the first-class function representation
166    /// from a pair of `(constrained, unconstrained)` to `(unconstrained, unconstrained)`
167    force_unconstrained: bool,

182/// Starting from the given `main` function, monomorphize the entire program,
183/// replacing all references to type variables and NamedGenerics with concrete
184/// types, duplicating definitions as necessary to do so.
185///
186/// Instead of iterating over every function, this pass starts with the main function
187/// and monomorphizes every function reachable from it via function calls and references.
188/// Thus, if a function is not used in the program, it will not be monomorphized.
189///
190/// Note that there is no requirement on the `main` function that can be passed into
191/// this function. Typically, this is the function named "main" in the source project,
192/// but it can also be, for example, an arbitrary test function for running `nargo test`.
193#[tracing::instrument(level = "trace", skip(main, interner))]
194pub fn monomorphize(
195    main: node_interner::FuncId,
196    interner: &mut NodeInterner,
197    force_unconstrained: bool,
198) -> Result<Program, MonomorphizationError> {
199    monomorphize_debug(main, interner, &DebugInstrumenter::default(), force_unconstrained)
200}
201
202/// A more general entry-point for the monomorphization pass containing an optional
203/// [DebugInstrumenter] which can be set to [DebugInstrumenter::default] in case it
204/// is not desired. If debugging is desired, additional function calls will be inserted
205/// to inspect values via debug functions.
206pub fn monomorphize_debug(
207    main: node_interner::FuncId,
208    interner: &mut NodeInterner,
209    debug_instrumenter: &DebugInstrumenter,
210    force_unconstrained: bool,
211) -> Result<Program, MonomorphizationError> {
212    let debug_type_tracker = DebugTypeTracker::build_from_debug_instrumenter(debug_instrumenter);
213    let mut monomorphizer = Monomorphizer::new(interner, debug_type_tracker, force_unconstrained);
214    monomorphizer.compile_main(main)?;
215
216    monomorphizer.process_queue()?;
217    Ok(monomorphizer.into_program())
218}

250    /// Pop the front of the function queue and monomorphize that function.
251    ///
252    /// Returns Ok(true) if there are still more jobs to process
253    /// Returns Ok(false) if there are no more jobs to process
254    /// Returns Err(_) if monomorphization encountered an error
255    pub fn process_next_job(&mut self) -> Result<bool, MonomorphizationError> {
256        let Some((next_fn_id, new_id, bindings, trait_method, is_unconstrained, location)) =
257            self.queue.pop_front()
258        else {
259            return Ok(false);
260        };
261
262        self.locals.clear();
263        self.in_unconstrained_function = is_unconstrained;
264
265        perform_instantiation_bindings(&bindings);
266        let interner = &self.interner;
267        let impl_bindings = perform_impl_bindings(interner, trait_method, next_fn_id, location)
268            .map_err(MonomorphizationError::InterpreterError)?;
269
270        self.function(next_fn_id, new_id, location)?;
271        undo_instantiation_bindings(impl_bindings);
272        undo_instantiation_bindings(bindings);
273
274        Ok(true)
275    }

277    /// Process any functions in the function queue to be monomorphized until
278    /// the queue is empty.
279    ///
280    /// Each function will push any new function-type combinations it encounters
281    /// during monomorphization so running the queue to completion will monomorphize
282    /// every function needed to run the program from the original entry point.
283    ///
284    /// Since this function does not fill the queue with any initial values, this
285    /// will need to be done beforehand by monomorphizing the entry point(s) beforehand.
286    pub fn process_queue(&mut self) -> Result<(), MonomorphizationError> {
287        while self.process_next_job()? {}
288        Ok(())
289    }

320    /// Collect all the finished functions and globals into a [Program].
321    ///
322    /// This will also run the ownership and proxies passes on the resulting
323    /// program.
324    pub fn into_program(self) -> Program {
325        let mut functions = vecmap(self.finished_functions, |(_, f)| f);
326        let globals = self.finished_globals.into_iter().collect::<BTreeMap<_, _>>();
327        let (debug_variables, debug_functions, debug_types) =
328            self.debug_type_tracker.extract_vars_and_types();
329
330        for f in functions.iter_mut() {
331            let is_acir_entry_point = !self.force_unconstrained && f.inline_type.is_entry_point();
332            f.is_entry_point = is_acir_entry_point || f.id == Program::main_id();
333        }
334
335        Program::new(
336            functions,
337            self.return_location,
338            globals,
339            debug_variables,
340            debug_functions,
341            debug_types,
342        )
343        .handle_ownership()
344        .create_foreign_proxies()
345    }

375    /// Retrieve the definition for the given function.
376    ///
377    /// If the given function has yet to be monomorphized, we'll create its new id now and return
378    /// that while queueing the full function to be monomorphized later.
379    fn lookup_function(
380        &mut self,
381        id: node_interner::FuncId,
382        expr_id: ExprId,
383        typ: &HirType,
384        turbofish_generics: &[HirType],
385        trait_method: Option<TraitItemId>,
386        evaluate_builtin: bool,
387    ) -> Result<Definition, MonomorphizationError> {
388        let typ = typ.follow_bindings();
389        let turbofish_generics = vecmap(turbofish_generics, |typ| typ.follow_bindings());
390        let is_unconstrained = self.is_unconstrained(id);
391
392        let definition = match self
393            .functions
394            .get(&(id, is_unconstrained))
395            .and_then(|by_func_type| by_func_type.get(&typ))
396            .and_then(|by_turbofish| by_turbofish.get(&turbofish_generics))
397        {
398            Some(id) => Definition::Function(*id),
399            None => {
400                // Function has not been monomorphized yet
401                let attributes = self.interner.function_attributes(&id);
402                match self.interner.function_meta(&id).kind {
403                    FunctionKind::LowLevel => {
404                        let attribute = attributes.function().expect("all low level functions must contain a function attribute which contains the opcode which it links to");
405                        let opcode = attribute.kind.foreign().expect(
406                            "ICE: function marked as foreign, but attribute kind does not match this",
407                        );
408                        Definition::LowLevel(opcode.to_string())
409                    }
410                    FunctionKind::Builtin => {
411                        let attribute = attributes.function().expect("all builtin functions must contain a function attribute which contains the opcode which it links to");
412                        let opcode = attribute.kind.builtin().expect(
413                            "ICE: function marked as builtin, but attribute kind does not match this",
414                        );
415                        let location = self.interner.expr_location(&expr_id);
416                        let opcode = opcode.to_string();
417
418                        if evaluate_builtin {
419                            match self.try_evaluate_builtin(
420                                &opcode,
421                                typ,
422                                turbofish_generics,
423                                is_unconstrained,
424                                id,
425                                location,
426                            )? {
427                                Some(id) => Definition::Function(id),
428                                None => Definition::Builtin(opcode),
429                            }
430                        } else {
431                            Definition::Builtin(opcode)
432                        }
433                    }
434                    FunctionKind::Normal | FunctionKind::TraitFunctionWithoutBody => {
435                        let id =
436                            self.queue_function(id, expr_id, typ, turbofish_generics, trait_method);
437                        Definition::Function(id)
438                    }
439                    FunctionKind::Oracle => {
440                        let attribute = attributes.function().expect("all oracle functions must contain a function attribute which contains the opcode which it links to");
441                        let opcode = attribute.kind.oracle().expect(
442                            "ICE: function marked as builtin, but attribute kind does not match this",
443                        );
444                        Definition::Oracle(opcode.to_string())
445                    }
446                }
447            }
448        };
449        Ok(definition)
450    }

480    /// Monomorphize the `main` function, ensuring it gets the ID expected by [Program::main_id].
481    ///
482    /// Sets the `return_location` expected by `into_program` later.
483    ///
484    /// Panics if some other function has already been monomorphized before.
485    pub fn compile_main(
486        &mut self,
487        main_id: node_interner::FuncId,
488    ) -> Result<(), MonomorphizationError> {
489        let new_main_id = self.next_function_id();
490        assert_eq!(new_main_id, Program::main_id(), "expected main to be monomorphized first");
491
492        let location = self.interner.function_meta(&main_id).location;
493        self.in_unconstrained_function = self.is_unconstrained(main_id);
494        self.function(main_id, new_main_id, location)?;
495
496        self.return_location =
497            self.interner.function(&main_id).block(self.interner).statements().last().and_then(
498                |x| match self.interner.statement(x) {
499                    HirStatement::Expression(id) => Some(self.interner.id_location(id)),
500                    _ => None,
501                },
502            );
503        Ok(())
504    }

506    /// Monomorphizes the given function.
507    ///
508    /// Expects any generics to already be bound by their bindings at this function's call site.
509    /// If this is not done, the relevant generic will either remain unbound (leading to a panic)
510    /// or, if the same generic is used further up the callstack (common with traits), it may have
511    /// an incorrect value. This incorrect type value will not be caught by monomorphization but
512    /// may or may not lead to panics later. Either way, it is best to only call this function
513    /// directly only when you know there should be no generics in your function, such as in
514    /// [Self::compile_main], or through a wrapper such as [Self::process_next_job] which will
515    /// handle the instantiation bindings for you.
516    pub fn function(
517        &mut self,
518        f: node_interner::FuncId,
519        id: FuncId,
520        location: Location,
521    ) -> Result<(), MonomorphizationError> {
522        if let Some((self_type, trait_id)) = self.interner.get_function_trait(&f) {
523            let the_trait = self.interner.get_trait(trait_id);
524            the_trait.self_type_typevar.force_bind(self_type);
525        }
526
527        let meta = self.interner.function_meta(&f).clone();
528
529        let modifiers = self.interner.function_modifiers(&f);
530        let name = self.interner.function_name(&f).to_owned();
531
532        if modifiers.is_comptime {
533            return Err(MonomorphizationError::ComptimeFnInRuntimeCode { name, location });
534        }
535
536        let body_expr_id = self.interner.function(&f).as_expr();
537        let body_return_type = self.interner.id_type(body_expr_id);
538        let return_type = match meta.return_type() {
539            Type::TraitAsType(..) => &body_return_type,
540            other => other,
541        };
542
543        let attributes = self.interner.function_attributes(&f);
544        let mut inline_type = InlineType::from(attributes);
545        let unconstrained = self.in_unconstrained_function;
546        if unconstrained {
547            inline_type = inline_type.into_unconstrained();
548        }
549        let is_fold = matches!(inline_type, InlineType::Fold);
550        let is_no_predicate = matches!(inline_type, InlineType::NoPredicates);
551
552        // We already checked the types are valid during elaboration. However, types behind
553        // generics (type variables) can't be known until monomorphization, so here we have
554        // to check again.
555        if is_fold || is_no_predicate {
556            for (pattern, typ, _visibility) in &meta.parameters.0 {
557                if let Some(invalid_type) = typ.non_inlined_function_input_validity() {
558                    let location = pattern.location();
559                    return Err(MonomorphizationError::InvalidTypeForEntryPoint {
560                        invalid_type,
561                        location,
562                    });
563                }
564            }
565
566            let output = true;
567            if let Some(invalid_type) = return_type.program_validity(output) {
568                let location = meta.return_type.location();
569                return Err(MonomorphizationError::InvalidTypeForEntryPoint {
570                    invalid_type,
571                    location,
572                });
573            }
574        }
575
576        // If `convert_type` fails here it is most likely because of generics at the
577        // call site after instantiating this function's type. So show the error there
578        // instead of at the function definition.
579        let return_type = Self::convert_type(return_type, location)?;
580        let return_visibility = meta.return_visibility;
581
582        let parameters = self.parameters(&meta.parameters)?;
583
584        let body = self.expr(body_expr_id)?;
585        let function = Function {
586            id,
587            name,
588            parameters,
589            body,
590            return_type,
591            return_visibility,
592            unconstrained,
593            inline_type,
594            is_entry_point: false,
595        };
596
597        self.push_function(id, function);
598        Ok(())
599    }

613        &mut self,
614        params: &Parameters,
615    ) -> Result<Vec<(LocalId, bool, String, ast::Type, Visibility)>, MonomorphizationError> {
616        let mut new_params = Vec::with_capacity(params.len());
617        for (parameter, typ, visibility) in &params.0 {
618            self.parameter(parameter, typ, visibility, &mut new_params)?;
619        }
620        Ok(new_params)
621    }

623    /// Monomorphize a single function parameters, potentially expanding it into multiple parameters
624    /// if it's a tuple/struct pattern, appending the new parameters to an accumulator.
625    fn parameter(
626        &mut self,
627        param: &HirPattern,
628        typ: &HirType,
629        visibility: &Visibility,
630        new_params: &mut Vec<(LocalId, bool, String, ast::Type, Visibility)>,
631    ) -> Result<(), MonomorphizationError> {
632        match param {
633            HirPattern::Identifier(ident) => {
634                let new_id = self.next_local_id();
635                let definition = self.interner.definition(ident.id);
636                let name = definition.name.clone();
637                let typ = Self::convert_type(typ, ident.location)?;
638                new_params.push((new_id, definition.mutable, name, typ, *visibility));
639                self.define_local(ident.id, new_id);
640            }
641            HirPattern::Mutable(pattern, _) => {
642                self.parameter(pattern, typ, visibility, new_params)?;
643            }
644            HirPattern::Tuple(fields, _) => {
645                let tuple_field_types = unwrap_tuple_type(typ);
646                assert_eq!(
647                    fields.len(),
648                    tuple_field_types.len(),
649                    "ICE: Unexpected number of tuple pattern fields"
650                );
651                for (field, typ) in fields.iter().zip(tuple_field_types) {
652                    self.parameter(field, &typ, visibility, new_params)?;
653                }
654            }
655            HirPattern::Struct(_, fields, location) => {
656                let struct_field_types = unwrap_struct_type(typ, *location)?;
657                assert_eq!(
658                    struct_field_types.len(),
659                    fields.len(),
660                    "ICE: Unexpected number of struct pattern fields"
661                );
662
663                let mut fields = btree_map(fields, |(name, field)| (name.to_string(), field));
664
665                // Iterate over `struct_field_types` since `unwrap_struct_type` will always
666                // return the fields in the order defined by the struct type.
667                for (field_name, field_type, _) in struct_field_types {
668                    let field = fields.remove(&field_name).unwrap_or_else(|| {
669                        unreachable!("Expected a field named '{field_name}' in the struct pattern")
670                    });
671
672                    self.parameter(field, &field_type, visibility, new_params)?;
673                }
674            }
675        }
676        Ok(())
677    }

684    /// Monomorphize an expression with a target type.
685    ///
686    /// This can be used when we have a type that contains bound type variables on the LHS,
687    /// and a type with unbound named generic on the RHS, and we don't want the unbound
688    /// types to be converted to default values.
689    fn expr_with_target_type(
690        &mut self,
691        expr: ExprId,
692        target_type: Option<&Type>,
693    ) -> Result<ast::Expression, MonomorphizationError> {
694        use ast::Expression::Literal;
695        use ast::Literal::*;
696
697        let expr = match self.interner.expression(&expr) {
698            HirExpression::Ident(ident, generics) => {
699                self.ident(ident, expr, generics, false, true)?
700            }
701            HirExpression::Literal(HirLiteral::Str(contents)) => Literal(Str(contents)),
702            HirExpression::Literal(HirLiteral::FmtStr(fragments, idents, _length)) => {
703                let fields = try_vecmap(idents, |ident| self.expr(ident))?;
704                Literal(FmtStr(
705                    fragments,
706                    fields.len() as u64,
707                    Box::new(ast::Expression::Tuple(fields)),
708                ))
709            }
710            HirExpression::Literal(HirLiteral::Bool(value)) => Literal(Bool(value)),
711            HirExpression::Literal(HirLiteral::Integer(value)) => {
712                let location = self.interner.id_location(expr);
713                let typ = Self::convert_type(&self.interner.id_type(expr), location)?;
714                Literal(Integer(value, typ, location))
715            }
716            HirExpression::Literal(HirLiteral::Array(array)) => match array {
717                HirArrayLiteral::Standard(array) => self.standard_array(expr, array, false)?,
718                HirArrayLiteral::Repeated { repeated_element, length } => {
719                    self.repeated_array(expr, repeated_element, length, false)?
720                }
721            },
722            HirExpression::Literal(HirLiteral::Vector(array)) => match array {
723                HirArrayLiteral::Standard(array) => self.standard_array(expr, array, true)?,
724                HirArrayLiteral::Repeated { repeated_element, length } => {
725                    self.repeated_array(expr, repeated_element, length, true)?
726                }
727            },
728            HirExpression::Literal(HirLiteral::Unit) => ast::Expression::Block(vec![]),
729            HirExpression::Block(block) => self.block(block.statements)?,
730            HirExpression::Unsafe(block) => self.block(block.statements)?,
731
732            HirExpression::Prefix(prefix) => {
733                let rhs = self.expr(prefix.rhs)?;
734                let location = self.interner.expr_location(&expr);
735
736                if self.interner.get_selected_impl_for_expression(expr).is_some() {
737                    // If an impl was selected for this prefix operator, replace it
738                    // with a method call to the appropriate trait impl method.
739                    let (function_type, ret) =
740                        self.interner.get_prefix_operator_type(expr, prefix.rhs);
741
742                    let method = prefix
743                        .trait_method_id
744                        .expect("ICE: missing trait method when impl was found");
745
746                    // `true` here to use the current runtime since we're immediately calling this
747                    // method in our current runtime.
748                    let func = self.resolve_trait_item_expr(expr, function_type, method, true)?;
749                    self.create_prefix_operator_impl_call(func, rhs, ret, location)?
750                } else {
751                    let operator = prefix.operator;
752                    let rhs = Box::new(rhs);
753                    let result_type = Self::convert_type(&self.interner.id_type(expr), location)?;
754                    ast::Expression::Unary(ast::Unary {
755                        operator,
756                        rhs,
757                        result_type,
758                        location,
759                        skip: prefix.skip,
760                    })
761                }
762            }
763
764            HirExpression::Infix(infix) => {
765                let lhs = self.expr(infix.lhs)?;
766                let rhs = self.expr(infix.rhs)?;
767                let operator = infix.operator.kind;
768                let location = self.interner.expr_location(&expr);
769                if self.interner.get_selected_impl_for_expression(expr).is_some() {
770                    // If an impl was selected for this infix operator, replace it
771                    // with a method call to the appropriate trait impl method.
772                    let (function_type, ret) =
773                        self.interner.get_infix_operator_type(infix.lhs, operator, expr);
774
775                    let method = infix.trait_method_id;
776
777                    let method = method.expect("ICE: Monomorphizer::expr: expected the infix operator method to be resolved by monomorphization");
778                    // `true` here to use the current runtime since we're immediately calling this
779                    // operator method in the current runtime.
780                    let func = self.resolve_trait_item_expr(expr, function_type, method, true)?;
781                    let operator = infix.operator;
782                    self.create_infix_operator_impl_call(func, lhs, operator, rhs, ret, location)?
783                } else {
784                    let lhs = Box::new(lhs);
785                    let rhs = Box::new(rhs);
786                    ast::Expression::Binary(ast::Binary { lhs, rhs, operator, location })
787                }
788            }
789
790            HirExpression::Index(index) => self.index(expr, index)?,
791
792            HirExpression::MemberAccess(access) => {
793                let field_index = self.interner.get_field_index(expr);
794                let expr = Box::new(self.expr(access.lhs)?);
795                ast::Expression::ExtractTupleField(expr, field_index)
796            }
797
798            HirExpression::Call(call) => self.function_call(call, expr)?,
799
800            HirExpression::Constrain(constrain) => {
801                let expr = self.expr(constrain.0)?;
802                let location = self.interner.expr_location(&constrain.0);
803                let assert_message = constrain
804                    .2
805                    .map(|assert_msg_expr| {
806                        self.expr(assert_msg_expr).map(|expr| {
807                            let typ = self.interner.id_type(assert_msg_expr).follow_bindings();
808                            let loc = self.interner.expr_location(&assert_msg_expr);
809                            (expr, typ, loc)
810                        })
811                    })
812                    .transpose()?;
813
814                if let Some((_, typ, location)) = assert_message.as_ref() {
815                    let check_msg_compat = |typ: &Type| {
816                        if !typ.is_message_compatible(true) {
817                            Err(MonomorphizationError::InvalidTypeInErrorMessage {
818                                typ: typ.to_string(),
819                                location: *location,
820                            })
821                        } else {
822                            Ok(())
823                        }
824                    };
825                    match typ {
826                        Type::FmtString(_, items) => match items.as_ref() {
827                            Type::Tuple(items) => {
828                                for item in items {
829                                    check_msg_compat(item)?;
830                                }
831                            }
832                            Type::Unit => {}
833                            other => {
834                                unreachable!("ICE: expected Tuple or Unit in fmtstr; got {other}")
835                            }
836                        },
837                        other => {
838                            check_msg_compat(other)?;
839                        }
840                    }
841                }
842
843                ast::Expression::Constrain(
844                    Box::new(expr),
845                    location,
846                    assert_message.map(|(msg, typ, _)| (msg, typ)).map(Box::new),
847                )
848            }
849
850            HirExpression::Cast(cast) => {
851                let location = self.interner.expr_location(&expr);
852                let typ = Self::convert_type(&cast.r#type, location)?;
853                let lhs = Box::new(self.expr(cast.lhs)?);
854                ast::Expression::Cast(ast::Cast { lhs, r#type: typ, location })
855            }
856
857            HirExpression::If(if_expr) => {
858                let condition = Box::new(self.expr(if_expr.condition)?);
859                let consequence = Box::new(self.expr(if_expr.consequence)?);
860                let else_ =
861                    if_expr.alternative.map(|alt| self.expr(alt)).transpose()?.map(Box::new);
862
863                let location = self.interner.expr_location(&expr);
864                let frontend_type = self.interner.id_type(expr);
865                let typ = Self::convert_type(&frontend_type, location)?;
866
867                if !self.in_unconstrained_function && frontend_type.contains_reference() {
868                    let typ = frontend_type.to_string();
869                    return Err(MonomorphizationError::ReferenceReturnedFromIfOrMatch {
870                        typ,
871                        location,
872                    });
873                }
874
875                ast::Expression::If(ast::If { condition, consequence, alternative: else_, typ })
876            }
877
878            HirExpression::Match(match_expr) => self.match_expr(match_expr, expr)?,
879
880            HirExpression::Tuple(fields) => {
881                let fields = try_vecmap(fields, |id| self.expr(id))?;
882                ast::Expression::Tuple(fields)
883            }
884            HirExpression::Constructor(constructor) => {
885                let typ = self.expr_or_target_type(expr, target_type);
886                self.constructor(constructor, expr, typ.as_ref())?
887            }
888
889            HirExpression::Lambda(lambda) => self.lambda(lambda, expr)?,
890
891            HirExpression::Error => unreachable!("Encountered Error node during monomorphization"),
892            HirExpression::Quote(_) => {
893                let location = self.interner.expr_location(&expr);
894                return Err(MonomorphizationError::ComptimeTypeInRuntimeCode {
895                    typ: "Quoted".to_string(),
896                    location,
897                });
898            }
899            HirExpression::Unquote(_) => {
900                unreachable!("unquote expression remaining in runtime code")
901            }
902            HirExpression::EnumConstructor(constructor) => {
903                let typ = self.expr_or_target_type(expr, target_type);
904                self.enum_constructor(constructor, expr, typ.as_ref())?
905            }
906        };
907
908        Ok(expr)
909    }

911    /// Return the target type if given, or the type of the expression if the target is empty.
912    ///
913    /// This is used in cases where the target type might have more bound generic type variables
914    /// than the expression, and unification is not feasible because the expression uses unbound
915    /// named generics, which don't get unified.
916    fn expr_or_target_type<'a>(
917        &self,
918        expr: ExprId,
919        target_type: Option<&'a Type>,
920    ) -> Cow<'a, HirType> {
921        match target_type {
922            Some(typ) => Cow::Borrowed(typ),
923            None => Cow::Owned(self.interner.id_type(expr)),
924        }
925    }

927    /// Monomorphize a "standard" array with all elements explicit, such as `[1, 5, 1, 3, 2]`.
928    /// This is in contrast to a repeated array such as `[0; 4]`.
929    fn standard_array(
930        &mut self,
931        array: ExprId,
932        array_elements: Vec<ExprId>,
933        is_vector: bool,
934    ) -> Result<ast::Expression, MonomorphizationError> {
935        let location = self.interner.expr_location(&array);
936        let typ = Self::convert_type(&self.interner.id_type(array), location)?;
937        let contents = try_vecmap(array_elements, |id| self.expr(id))?;
938        if is_vector {
939            Ok(ast::Expression::Literal(ast::Literal::Vector(ast::ArrayLiteral { contents, typ })))
940        } else {
941            Ok(ast::Expression::Literal(ast::Literal::Array(ast::ArrayLiteral { contents, typ })))
942        }
943    }

946    fn repeated_array(
947        &mut self,
948        array: ExprId,
949        repeated_element: ExprId,
950        length: HirType,
951        is_vector: bool,
952    ) -> Result<ast::Expression, MonomorphizationError> {
953        let location = self.interner.expr_location(&array);
954        let typ = Self::convert_type(&self.interner.id_type(array), location)?;
955
956        let length = length.evaluate_to_u32(location).map_err(|err| {
957            let location = self.interner.expr_location(&array);
958            MonomorphizationError::UnknownArrayLength { location, err }
959        })?;
960

962
963        Ok(ast::Expression::Literal(ast::Literal::Repeated {
964            element: Box::new(element),
965            length,
966            is_vector,
967            typ,
968        }))
969    }

985    fn statement(&mut self, id: StmtId) -> Result<ast::Expression, MonomorphizationError> {
986        match self.interner.statement(&id) {
987            HirStatement::Let(let_statement) => self.let_statement(let_statement),
988            HirStatement::Assign(assign) => self.assign(assign),
989            HirStatement::For(for_loop) => {
990                let start = self.expr(for_loop.start_range)?;
991                let end = self.expr(for_loop.end_range)?;
992                let index_variable = self.next_local_id();
993                self.define_local(for_loop.identifier.id, index_variable);
994
995                let block = Box::new(self.expr(for_loop.block)?);
996                let index_location = for_loop.identifier.location;
997                let index_type = self.interner.id_type(for_loop.start_range);
998                let index_type = Self::convert_type(&index_type, index_location)?;
999
1000                Ok(ast::Expression::For(ast::For {
1001                    index_variable,
1002                    index_name: self.interner.definition_name(for_loop.identifier.id).to_owned(),
1003                    index_type,
1004                    start_range: Box::new(start),
1005                    end_range: Box::new(end),
1006                    inclusive: for_loop.inclusive,
1007                    start_range_location: self.interner.expr_location(&for_loop.start_range),
1008                    end_range_location: self.interner.expr_location(&for_loop.end_range),
1009                    block,
1010                }))
1011            }
1012            HirStatement::Loop(block) => {
1013                let block = Box::new(self.expr(block)?);
1014                Ok(ast::Expression::Loop(block))
1015            }
1016            HirStatement::While(condition, body) => {
1017                let condition = Box::new(self.expr(condition)?);
1018                let body = Box::new(self.expr(body)?);
1019                Ok(ast::Expression::While(While { condition, body }))
1020            }
1021            HirStatement::Expression(expr) => self.expr(expr),
1022            HirStatement::Semi(expr) => {
1023                self.expr(expr).map(|expr| ast::Expression::Semi(Box::new(expr)))
1024            }
1025            HirStatement::Break => Ok(ast::Expression::Break),
1026            HirStatement::Continue => Ok(ast::Expression::Continue),
1027            HirStatement::Error => unreachable!(),
1028
1029            // All `comptime` statements & expressions should be removed before runtime.
1030            HirStatement::Comptime(_) => unreachable!("comptime statement in runtime code"),
1031        }
1032    }

1043    fn constructor(
1044        &mut self,
1045        constructor: HirConstructorExpression,
1046        id: ExprId,
1047        typ: &Type,
1048    ) -> Result<ast::Expression, MonomorphizationError> {
1049        let location = self.interner.expr_location(&id);
1050
1051        let field_types = unwrap_struct_type(typ, location)?;
1052
1053        let field_type_map =
1054            btree_map(&field_types, |(name, typ, _)| (name.to_string(), typ.clone()));
1055
1056        // Create let bindings for each field value first to preserve evaluation order before
1057        // they are reordered and packed into the resulting tuple
1058        let mut field_vars = BTreeMap::new();
1059        let mut new_exprs = Vec::with_capacity(constructor.fields.len());
1060
1061        for (field_name, expr_id) in constructor.fields {
1062            let new_id = self.next_local_id();
1063            let field_type = field_type_map.get(field_name.as_str()).unwrap();
1064            let location = self.interner.expr_location(&expr_id);
1065            let typ = Self::convert_type(field_type, location)?;
1066
1067            field_vars.insert(field_name.to_string(), (new_id, typ));
1068            let expression = Box::new(self.expr(expr_id)?);
1069
1070            new_exprs.push(ast::Expression::Let(ast::Let {
1071                id: new_id,
1072                mutable: false,
1073                name: field_name.into_string(),
1074                expression,
1075            }));
1076        }
1077
1078        // We must ensure the tuple created from the variables here matches the order
1079        // of the fields as defined in the type. To do this, we iterate over field_types,
1080        // rather than field_type_map which is a sorted BTreeMap.
1081        let field_idents = vecmap(field_types, |(name, _, _)| {
1082            let (id, typ) = field_vars.remove(&name).unwrap_or_else(|| {
1083                unreachable!("Expected field {name} to be present in constructor for {typ}")
1084            });
1085
1086            let definition = Definition::Local(id);
1087            let mutable = false;
1088            ast::Expression::Ident(ast::Ident {
1089                definition,
1090                mutable,
1091                location: None,
1092                name,
1093                typ,
1094                id: self.next_ident_id(),
1095            })
1096        });
1097
1098        // Finally we can return the created Tuple from the new block
1099        new_exprs.push(ast::Expression::Tuple(field_idents));
1100        Ok(ast::Expression::Block(new_exprs))
1101    }

1103    /// For an enum like:
1104    /// ```text
1105    /// enum Foo {
1106    ///    A(i32, u32),
1107    ///    B(Field),
1108    ///    C
1109    /// }
1110    /// ```
1111    /// this will translate the call `Foo::A(1, 2)` into `(0, (1, 2), (0,), ())` where
1112    /// the first field `0` is the tag value, the second is `A`, third is `B`, and fourth is `C`.
1113    /// Each variant that isn't the desired variant has zeroed values filled in for its data.
1114    fn enum_constructor(
1115        &mut self,
1116        constructor: HirEnumConstructorExpression,
1117        id: ExprId,
1118        typ: &Type,
1119    ) -> Result<ast::Expression, MonomorphizationError> {
1120        let location = self.interner.expr_location(&id);
1121        let variants = unwrap_enum_type(typ, location)?;
1122
1123        let tag_value = FieldElement::from(constructor.variant_index);
1124        let tag_value = SignedField::positive(tag_value);
1125        let tag = ast::Literal::Integer(tag_value, ast::Type::Field, location);
1126        let mut fields = vec![ast::Expression::Literal(tag)];
1127
1128        // Fill in each field of the translated enum tuple.
1129        // For most fields this will be simply `std::mem::zeroed::<T>()`,
1130        // but for the given variant we just pack all the arguments into a tuple for that field.
1131        for (i, (_, arg_types)) in variants.into_iter().enumerate() {
1132            let args = if i == constructor.variant_index {
1133                try_vecmap(&constructor.arguments, |arg| self.expr(*arg))
1134            } else {
1135                try_vecmap(arg_types, |typ| {
1136                    let typ = Self::convert_type(&typ, location)?;
1137                    Ok(self.zeroed_value_of_type(&typ, location))
1138                })
1139            }?;
1140            fields.push(ast::Expression::Tuple(args));
1141        }
1142
1143        Ok(ast::Expression::Tuple(fields))
1144    }

1154    fn unpack_pattern(
1155        &mut self,
1156        pattern: HirPattern,
1157        value: ast::Expression,
1158        typ: &HirType,
1159    ) -> Result<ast::Expression, MonomorphizationError> {
1160        match pattern {
1161            HirPattern::Identifier(ident) => {
1162                let new_id = self.next_local_id();
1163                self.define_local(ident.id, new_id);
1164                let definition = self.interner.definition(ident.id);
1165
1166                Ok(ast::Expression::Let(ast::Let {
1167                    id: new_id,
1168                    mutable: definition.mutable,
1169                    name: definition.name.clone(),
1170                    expression: Box::new(value),
1171                }))
1172            }
1173            HirPattern::Mutable(pattern, _) => self.unpack_pattern(*pattern, value, typ),
1174            HirPattern::Tuple(patterns, _) => {
1175                let fields = unwrap_tuple_type(typ);
1176                self.unpack_tuple_pattern(value, patterns.into_iter().zip(fields), typ)
1177            }
1178            HirPattern::Struct(_, patterns, location) => {
1179                let fields = unwrap_struct_type(typ, location)?;
1180                assert_eq!(patterns.len(), fields.len());
1181
1182                let mut patterns =
1183                    btree_map(patterns, |(name, pattern)| (name.into_string(), pattern));
1184
1185                // We iterate through the type's fields to match the order defined in the struct type
1186                let patterns_iter = fields.into_iter().map(|(field_name, field_type, _)| {
1187                    let pattern = patterns.remove(&field_name).unwrap();
1188                    (pattern, field_type)
1189                });
1190
1191                self.unpack_tuple_pattern(value, patterns_iter, typ)
1192            }
1193        }
1194    }

1196    fn unpack_tuple_pattern(
1197        &mut self,
1198        value: ast::Expression,
1199        fields: impl Iterator<Item = (HirPattern, HirType)>,
1200        tuple_type: &Type,
1201    ) -> Result<ast::Expression, MonomorphizationError> {
1202        let fresh_id = self.next_local_id();
1203
1204        let mut definitions = vec![ast::Expression::Let(ast::Let {
1205            id: fresh_id,
1206            mutable: false,
1207            name: "_".into(),
1208            expression: Box::new(value),
1209        })];
1210
1211        for (i, (field_pattern, field_type)) in fields.into_iter().enumerate() {
1212            let location = field_pattern.location();
1213            let mutable = false;
1214            let definition = Definition::Local(fresh_id);
1215            let name = i.to_string();
1216
1217            let typ = Self::convert_type(tuple_type, location)?;
1218            let location = Some(location);
1219            let id = self.next_ident_id();
1220            let new_rhs =
1221                ast::Expression::Ident(ast::Ident { location, mutable, definition, name, typ, id });
1222
1223            let new_rhs = ast::Expression::ExtractTupleField(Box::new(new_rhs), i);
1224            let new_expr = self.unpack_pattern(field_pattern, new_rhs, &field_type)?;
1225            definitions.push(new_expr);
1226        }
1227
1228        Ok(ast::Expression::Block(definitions))
1229    }

1272    fn ident(
1273        &mut self,
1274        ident: HirIdent,
1275        expr_id: ExprId,
1276        generics: Option<Vec<HirType>>,
1277        // If set and this is a function value, only monomorphize the function for the
1278        // constrainedness given by `self.in_unconstrained_function` rather than returning a tuple
1279        // of both (constrained, unconstrained). This is used only as an optimization to avoid
1280        // unnecessary monomorphization when calling a known function.
1281        use_current_runtime: bool,
1282        // If true, evaluate some builtins to function values. This is disabled when code-generating
1283        // the function in a function call since we can avoid creating a new function and instead
1284        // inline the body directly which keeps some minimal SSA pass tests working.
1285        evaluate_builtin: bool,
1286    ) -> Result<ast::Expression, MonomorphizationError> {
1287        let typ = self.interner.id_type(expr_id);
1288
1289        if let ImplKind::TraitItem(item) = ident.impl_kind {
1290            return self.resolve_trait_item_expr(expr_id, typ, item.id(), use_current_runtime);
1291        }
1292
1293        // Ensure all instantiation bindings are bound.
1294        // This ensures even unused type variables like `fn foo<T>() {}` have concrete types
1295        if let Some(bindings) = self.interner.try_get_instantiation_bindings(expr_id) {
1296            for (_, kind, binding) in bindings.values() {
1297                match kind {
1298                    Kind::Any => (),
1299                    Kind::Normal => (),
1300                    Kind::Integer => (),
1301                    Kind::IntegerOrField => (),
1302                    Kind::Numeric(typ) => Self::check_type(typ, ident.location)?,
1303                }
1304                Self::check_type(binding, ident.location)?;
1305            }
1306        }
1307
1308        let definition = self.interner.definition(ident.id);
1309        match &definition.kind {
1310            DefinitionKind::Function(func_id) => {
1311                let mutable = definition.mutable;
1312                let name = definition.name.clone();
1313                let generics = generics.clone();
1314                let func_id = *func_id;
1315
1316                // Functions are represented as a pair of their constrained and unconstrained versions
1317                self.monomorphize_constrained_and_unconstrained(
1318                    use_current_runtime,
1319                    self.force_unconstrained,
1320                    |this| {
1321                        this.function_reference(
1322                            mutable,
1323                            name,
1324                            ident.location,
1325                            func_id,
1326                            expr_id,
1327                            &typ,
1328                            generics,
1329                            evaluate_builtin,
1330                        )
1331                    },
1332                )
1333            }
1334            DefinitionKind::Global(global_id) => {
1335                self.global_ident(*global_id, definition.name.clone(), &typ, ident.location)
1336            }
1337            DefinitionKind::Local(_) => match self.lookup_captured_expr(ident.id) {
1338                Some(expr) => Ok(expr),
1339                None => {
1340                    let Some(ident) = self.local_ident(&ident, &typ)? else {
1341                        let location = self.interner.id_location(expr_id);
1342                        let message = "ICE: Variable not found during monomorphization";
1343                        return Err(MonomorphizationError::InternalError { location, message });
1344                    };
1345                    Ok(ast::Expression::Ident(ident))
1346                }
1347            },
1348            DefinitionKind::NumericGeneric(type_variable, numeric_typ) => {
1349                let location = self.interner.id_location(expr_id);
1350                let value = Type::TypeVariable(type_variable.clone());
1351                self.numeric_generic(value, numeric_typ.as_ref().clone(), typ, location)
1352            }
1353            DefinitionKind::AssociatedConstant(trait_impl_id, name) => {
1354                let location = ident.location;
1355                let associated_types = self.interner.get_associated_types_for_impl(*trait_impl_id);
1356                let associated_type = associated_types
1357                    .iter()
1358                    .find(|typ| typ.name.as_str() == name)
1359                    .expect("Expected to find associated type");
1360                let Kind::Numeric(numeric_type) = associated_type.typ.kind() else {
1361                    unreachable!("Expected associated type to be numeric");
1362                };
1363                match associated_type
1364                    .typ
1365                    .evaluate_to_signed_field(&associated_type.typ.kind(), location)
1366                {
1367                    Ok(value) => {
1368                        let typ = Self::convert_type(&numeric_type, location)?;
1369                        Ok(ast::Expression::Literal(ast::Literal::Integer(value, typ, location)))
1370                    }
1371                    Err(err) => Err(MonomorphizationError::CannotComputeAssociatedConstant {
1372                        name: name.clone(),
1373                        err,
1374                        location,
1375                    }),
1376                }
1377            }
1378        }
1379    }

1381    /// Create a function value for the given function (not an actual &mut reference to a function).
1382    /// Note that this will refer to a single constrained or unconstrained version of the given
1383    /// function, it will never return a tuple of (constrained, unconstrained), but may return a
1384    /// tuple if the function is a closure.
1385    #[allow(clippy::too_many_arguments)]
1386    fn function_reference(
1387        &mut self,
1388        mutable: bool,
1389        name: String,
1390        location: Location,
1391        func_id: node_interner::FuncId,
1392        expr_id: ExprId,
1393        typ: &Type,
1394        generics: Option<Vec<HirType>>,
1395        evaluate_builtin: bool,
1396    ) -> Result<ast::Expression, MonomorphizationError> {
1397        let definition = self.lookup_function(
1398            func_id,
1399            expr_id,
1400            typ,
1401            &generics.unwrap_or_default(),
1402            None,
1403            evaluate_builtin,
1404        )?;
1405        let typ = Self::convert_type(typ, location)?;
1406        let location = Some(location);
1407        let id = self.next_ident_id();
1408        let ident = ast::Ident { location, mutable, definition, name, typ: typ.clone(), id };
1409        let ident_expression = ast::Expression::Ident(ident);
1410
1411        if self.is_closure_type(&typ) {
1412            let ident_clone = Box::new(ident_expression.clone());
1413            let function = ast::Expression::ExtractTupleField(ident_clone, 0);
1414            let env = ast::Expression::ExtractTupleField(Box::new(ident_expression), 1);
1415            Ok(ast::Expression::Tuple(vec![function, env]))
1416        } else {
1417            Ok(ident_expression)
1418        }
1419    }

1421    /// Monomorphize a numeric generic as a numeric constant.
1422    /// Expects arguments to correspond to `let N: $expected_type = $value;`
1423    fn numeric_generic(
1424        &self,
1425        value: Type,
1426        expected_type: Type,
1427        expr_type: Type,
1428        location: Location,
1429    ) -> Result<ast::Expression, MonomorphizationError> {
1430        let expected_kind = Kind::Numeric(Box::new(expected_type.clone()));
1431        let value = value
1432            .evaluate_to_signed_field(&expected_kind, location)
1433            .map_err(|err| MonomorphizationError::UnknownArrayLength { err, location })?;
1434
1435        let expr_kind = Kind::Numeric(Box::new(expr_type.clone()));
1436        if !expected_kind.unifies(&expr_kind) {
1437            let message = "ICE: Generic's kind does not match expected type";
1438            return Err(MonomorphizationError::InternalError { location, message });
1439        }
1440
1441        let typ = Self::convert_type(&expected_type, location)?;
1442        Ok(ast::Expression::Literal(ast::Literal::Integer(value, typ, location)))
1443    }

1445    fn global_ident(
1446        &mut self,
1447        global_id: node_interner::GlobalId,
1448        name: String,
1449        typ: &HirType,
1450        location: Location,
1451    ) -> Result<ast::Expression, MonomorphizationError> {
1452        let global = self.interner.get_global(global_id);
1453        let id = global.id;
1454
1455        // Follow bindings otherwise it's not safe to use it as a hash map key.
1456        let typ = typ.follow_bindings();
1457
1458        let expr = if let Some(seen_global) = self.globals.get(&id).and_then(|m| m.get(&typ)) {
1459            let typ = Self::convert_type(&typ, location)?;
1460            let ident = ast::Ident {
1461                location: Some(location),
1462                definition: Definition::Global(*seen_global),
1463                mutable: false,
1464                name,
1465                typ,
1466                id: self.next_ident_id(),
1467            };
1468            ast::Expression::Ident(ident)
1469        } else {
1470            // Globals have been evaluated with the comptime interpreter. Convert that value to HIR.
1471            let (expr, contains_function) = if let GlobalValue::Resolved(value) =
1472                global.value.clone()
1473            {
1474                let contains_function = value.contains_function_or_closure();
1475                let expr = value
1476                    .into_runtime_hir_expression(self.interner, global.location)
1477                    .map_err(MonomorphizationError::InterpreterError)?;
1478                (expr, contains_function)
1479            } else {
1480                unreachable!(
1481                    "All global values should be resolved at compile time and before monomorphization"
1482                );
1483            };
1484
1485            // The type of the expression on the RHS itself might be a `Forall` and/or contain unbound `NamedGeneric`s,
1486            // while the type of the type on the LHS has bound `TypeVariable` variables. We cannot bind them,
1487            // because unbound `NamedGeneric`s don't unify with bound `TypeVariable`s, still we want to monomorphize
1488            // into expression specific to the LHS, so we are using it as the target type.
1489            let expr = self.expr_with_target_type(expr, Some(&typ))?;
1490
1491            // Globals are meant to be computed at compile time and are stored in their own context to be shared across functions.
1492            // Closures are defined as normal functions among all SSA functions and later need to be defunctionalized.
1493            // Thus, this means we would have to re-define any global closures.
1494            // The effect of defunctionalization would be the same if we were redefining a global closure or a local closure
1495            // just with an extra step of indirection through a global variable.
1496            // For simplicity, we chose to instead inline closures at their callsite as we do not expect
1497            // placing a closure in the global context to change the final result of the program.
1498            if !contains_function {
1499                let new_id = self.next_global_id();
1500                self.globals.entry(id).or_default().insert(typ.clone(), new_id);
1501                let typ = Self::convert_type(&typ, location)?;
1502                self.finished_globals.insert(new_id, (name.clone(), typ.clone(), expr));
1503                let ident = ast::Ident {
1504                    location: Some(location),
1505                    definition: Definition::Global(new_id),
1506                    mutable: false,
1507                    name,
1508                    typ,
1509                    id: self.next_ident_id(),
1510                };
1511                ast::Expression::Ident(ident)
1512            } else {
1513                expr
1514            }
1515        };
1516        Ok(expr)
1517    }

1519    /// Estimates the complexity of a type.
1520    /// This is roughly the number of "nodes" in the type tree.
1521    ///
1522    /// To prevent stack overflow on deeply nested types, we stop recursion when
1523    /// accumulated complexity exceeds MAX_TYPE_COMPLEXITY.
1524    fn type_complexity(typ: &HirType) -> usize {
1525        Self::type_complexity_inner(typ, 0)
1526    }
1527
1528    fn type_complexity_inner(typ: &HirType, accumulated: usize) -> usize {
1529        // Early return if accumulated complexity exceeds the limit,
1530        // this avoids stack overflow due to the recursive nature of this computation
1531        if accumulated > MAX_TYPE_COMPLEXITY {
1532            return accumulated + 1;
1533        }
1534
1535        let typ = typ.follow_bindings_shallow();
1536        match typ.as_ref() {
1537            HirType::Tuple(fields) => {
1538                let mut complexity = accumulated + 1;
1539                for field in fields {
1540                    complexity = Self::type_complexity_inner(field, complexity);
1541                }
1542                complexity
1543            }
1544            HirType::Array(_len, elem_typ) => {
1545                Self::type_complexity_inner(elem_typ, accumulated + 1)
1546            }
1547            HirType::DataType(_def, generics) => {
1548                let mut complexity = accumulated + 1;
1549                for generic in generics {
1550                    complexity = Self::type_complexity_inner(generic, complexity);
1551                }
1552                complexity
1553            }
1554            HirType::Function(args, ret, env, _) => {
1555                let mut complexity = accumulated + 1;
1556                for arg in args {
1557                    complexity = Self::type_complexity_inner(arg, complexity);
1558                }
1559                complexity = Self::type_complexity_inner(ret, complexity);
1560                Self::type_complexity_inner(env, complexity)
1561            }
1562            HirType::Reference(inner, _) => Self::type_complexity_inner(inner, accumulated + 1),
1563            HirType::Alias(_, generics) => {
1564                let mut complexity = accumulated + 1;
1565                for generic in generics {
1566                    complexity = Self::type_complexity_inner(generic, complexity);
1567                }
1568                complexity
1569            }
1570            // Simple types
1571            _ => accumulated + 1,
1572        }
1573    }

1583    /// Converts a [HirType] into a [ast::Type].
1584    ///
1585    /// Returns an error if the type was invalid somehow. For example, if the type contains:
1586    /// - an array with a size that does not evaluate to a positive integer
1587    /// - an unbound type variable without a default value
1588    /// - a nested vector
1589    /// - a compile-time only type (these should all be evaluated beforehand and thus never monomorphized)
1590    /// - an infinitely recursive type
1591    /// - a failed checked-cast
1592    fn convert_type_helper(
1593        typ: &HirType,
1594        location: Location,
1595        seen_types: &mut HashSet<Type>,
1596    ) -> Result<ast::Type, MonomorphizationError> {
1597        let complexity = Self::type_complexity(typ);
1598        if complexity > MAX_TYPE_COMPLEXITY {
1599            return Err(MonomorphizationError::ComplexType {
1600                complexity,
1601                max_complexity: MAX_TYPE_COMPLEXITY,
1602                location,
1603            });
1604        }
1605
1606        let typ = typ.follow_bindings_shallow();
1607        Ok(match typ.as_ref() {
1608            HirType::FieldElement => ast::Type::Field,
1609            HirType::Integer(sign, bits) => ast::Type::Integer(*sign, *bits),
1610            HirType::Bool => ast::Type::Bool,
1611            HirType::String(size) => {
1612                let size = match size.evaluate_to_u32(location) {
1613                    Ok(size) => size,
1614                    Err(err) => {
1615                        return Err(MonomorphizationError::UnknownArrayLength { location, err });
1616                    }
1617                };
1618                ast::Type::String(size)
1619            }
1620            HirType::FmtString(size, fields) => {
1621                let size = match size.evaluate_to_u32(location) {
1622                    Ok(size) => size,
1623                    Err(err) => {
1624                        return Err(MonomorphizationError::UnknownArrayLength { location, err });
1625                    }
1626                };
1627                let fields =
1628                    Box::new(Self::convert_type_helper(fields.as_ref(), location, seen_types)?);
1629                ast::Type::FmtString(size, fields)
1630            }
1631            HirType::Unit => ast::Type::Unit,
1632            HirType::Array(length, element) => {
1633                if element.contains_vector() {
1634                    return Err(MonomorphizationError::NestedVectors { location });
1635                }
1636                let element =
1637                    Box::new(Self::convert_type_helper(element.as_ref(), location, seen_types)?);
1638                let length = match length.evaluate_to_u32(location) {
1639                    Ok(length) => length,
1640                    Err(err) => {
1641                        return Err(MonomorphizationError::UnknownArrayLength { location, err });
1642                    }
1643                };
1644                ast::Type::Array(length, element)
1645            }
1646            HirType::Vector(element) => {
1647                if element.contains_vector() {
1648                    return Err(MonomorphizationError::NestedVectors { location });
1649                }
1650                let element =
1651                    Box::new(Self::convert_type_helper(element.as_ref(), location, seen_types)?);
1652                ast::Type::Vector(element)
1653            }
1654            HirType::TraitAsType(..) => {
1655                unreachable!("All TraitAsType should be replaced before calling convert_type");
1656            }
1657            HirType::NamedGeneric(NamedGeneric { type_var, .. }) => {
1658                if let TypeBinding::Bound(binding) = &*type_var.borrow() {
1659                    return Self::convert_type_helper(binding, location, seen_types);
1660                }
1661                // This used to default to Field, but doing so could result in an invalid SSA.
1662                return Err(MonomorphizationError::NoDefaultType { location });
1663            }
1664
1665            HirType::CheckedCast { from, to } => {
1666                Self::check_checked_cast(from, to, location)?;
1667                Self::convert_type_helper(to, location, seen_types)?
1668            }
1669
1670            HirType::TypeVariable(binding) => {
1671                let input_type = typ.as_ref().clone();
1672                if !seen_types.insert(input_type.clone()) {
1673                    let typ = input_type;
1674                    return Err(MonomorphizationError::RecursiveType { typ, location });
1675                }
1676
1677                let type_var_kind = match &*binding.borrow() {
1678                    TypeBinding::Bound(binding) => {
1679                        let typ = Self::convert_type_helper(binding, location, seen_types);
1680                        seen_types.remove(&input_type);
1681                        return typ;
1682                    }
1683                    TypeBinding::Unbound(_, type_var_kind) => type_var_kind.clone(),
1684                };
1685
1686                // Default any remaining unbound type variables.
1687                // This should only happen if the variable in question is unused
1688                // and within a larger generic type.
1689                let default = match type_var_kind.default_type() {
1690                    Some(typ) => typ,
1691                    None => {
1692                        return Err(MonomorphizationError::NoDefaultType { location });
1693                    }
1694                };
1695
1696                let monomorphized_default =
1697                    Self::convert_type_helper(&default, location, seen_types)?;
1698                binding.bind(default);
1699                monomorphized_default
1700            }
1701
1702            HirType::DataType(def, args) => {
1703                // Not all generic arguments may be used in a datatype's fields so we have to check
1704                // the arguments as well as the fields in case any need to be defaulted or are unbound.
1705                for arg in args {
1706                    Self::check_type(arg, location)?;
1707                }
1708
1709                let input_type = typ.as_ref().clone();
1710                if !seen_types.insert(input_type.clone()) {
1711                    let typ = input_type;
1712                    return Err(MonomorphizationError::RecursiveType { typ, location });
1713                }
1714
1715                let def = def.borrow();
1716                if let Some(fields) = def.get_fields(args) {
1717                    let fields = try_vecmap(fields, |(_, field, _)| {
1718                        Self::convert_type_helper(&field, location, seen_types)
1719                    })?;
1720
1721                    seen_types.remove(&input_type);
1722                    ast::Type::Tuple(fields)
1723                } else if let Some(variants) = def.get_variants(args) {
1724                    // Enums are represented as (tag, variant1, variant2, .., variantN)
1725                    let mut fields = vec![ast::Type::Field];
1726                    for (_, variant_fields) in variants {
1727                        let variant_fields = try_vecmap(variant_fields, |typ| {
1728                            Self::convert_type_helper(&typ, location, seen_types)
1729                        })?;
1730                        fields.push(ast::Type::Tuple(variant_fields));
1731                    }
1732                    seen_types.remove(&input_type);
1733                    ast::Type::Tuple(fields)
1734                } else {
1735                    unreachable!("Data type has no body")
1736                }
1737            }
1738
1739            HirType::Alias(def, args) => {
1740                // Similar to the struct case above: generics of an alias might not end up being
1741                // used in the type that is aliased.
1742                for arg in args {
1743                    Self::check_type(arg, location)?;
1744                }
1745
1746                Self::convert_type_helper(&def.borrow().get_type(args), location, seen_types)?
1747            }
1748
1749            HirType::Tuple(fields) => {
1750                let fields =
1751                    try_vecmap(fields, |x| Self::convert_type_helper(x, location, seen_types))?;
1752                ast::Type::Tuple(fields)
1753            }
1754
1755            HirType::Function(args, ret, env, _unconstrained) => {
1756                let args =
1757                    try_vecmap(args, |x| Self::convert_type_helper(x, location, seen_types))?;
1758                let ret = Box::new(Self::convert_type_helper(ret, location, seen_types)?);
1759                let env = Self::convert_type_helper(env, location, seen_types)?;
1760
1761                let make_function = |is_unconstrained, args, ret, env| {
1762                    use ast::Type::*;
1763                    match &env {
1764                        Unit => Function(args, ret, Box::new(env), is_unconstrained),
1765                        Tuple(_) => Tuple(vec![
1766                            env.clone(),
1767                            Function(args, ret, Box::new(env), is_unconstrained),
1768                        ]),
1769                        _ => {
1770                            unreachable!(
1771                                "internal Type::Function env should be either a Unit or a Tuple, not {env}"
1772                            )
1773                        }
1774                    }
1775                };
1776
1777                // functions are represented as a tuple of both
1778                // a constrained and unconstrained version of the same function.
1779                let constrained = make_function(false, args.clone(), ret.clone(), env.clone());
1780                let unconstrained = make_function(true, args, ret, env);
1781                ast::Type::Tuple(vec![constrained, unconstrained])
1782            }
1783
1784            // Lower both mutable & immutable references to the same reference type
1785            HirType::Reference(element, mutable) => {
1786                let element = Self::convert_type_helper(element, location, seen_types)?;
1787                ast::Type::Reference(Box::new(element), *mutable)
1788            }
1789
1790            HirType::Forall(_, _) | HirType::Constant(..) | HirType::InfixExpr(..) => {
1791                unreachable!("Unexpected type {typ} found")
1792            }
1793            HirType::Error => {
1794                let message = "Unexpected Type::Error found during monomorphization";
1795                return Err(MonomorphizationError::InternalError { message, location });
1796            }
1797            HirType::Quoted(typ) => {
1798                let typ = typ.to_string();
1799                return Err(MonomorphizationError::ComptimeTypeInRuntimeCode { typ, location });
1800            }
1801        })

1804    /// Similar to `convert_type` but only checks for errors and does not actually convert to a
1805    /// [ast::Type].
1806    ///
1807    /// This function also does not recur completely on a type (for example, it does
1808    /// not check fields of a struct) to prevent infinite recursion.
1809    fn check_type(typ: &HirType, location: Location) -> Result<(), MonomorphizationError> {
1810        let typ = typ.follow_bindings_shallow();
1811        match typ.as_ref() {
1812            HirType::FieldElement
1813            | HirType::Integer(..)
1814            | HirType::Bool
1815            | HirType::String(..)
1816            | HirType::Unit
1817            | HirType::TraitAsType(..)
1818            | HirType::Forall(_, _)
1819            | HirType::Error
1820            | HirType::Quoted(_) => Ok(()),
1821            HirType::Constant(_value, kind) => {
1822                if kind.is_error() {
1823                    Err(MonomorphizationError::UnknownConstant { location })
1824                } else {
1825                    Ok(())
1826                }
1827            }
1828            HirType::CheckedCast { from, to } => {
1829                Self::check_checked_cast(from, to, location)?;
1830                Self::check_type(to, location)
1831            }
1832
1833            HirType::FmtString(_size, fields) => Self::check_type(fields.as_ref(), location),
1834            HirType::Array(_length, element) => {
1835                if element.contains_vector() {
1836                    return Err(MonomorphizationError::NestedVectors { location });
1837                }
1838                Self::check_type(element.as_ref(), location)
1839            }
1840            HirType::Vector(element) => {
1841                if element.contains_vector() {
1842                    return Err(MonomorphizationError::NestedVectors { location });
1843                }
1844                Self::check_type(element.as_ref(), location)
1845            }
1846            HirType::NamedGeneric(NamedGeneric { type_var, .. }) => {
1847                if let TypeBinding::Bound(binding) = &*type_var.borrow() {
1848                    return Self::check_type(binding, location);
1849                }
1850
1851                Ok(())
1852            }
1853            HirType::TypeVariable(binding) => {
1854                let type_var_kind = match &*binding.borrow() {
1855                    TypeBinding::Bound(binding) => {
1856                        return Self::check_type(binding, location);
1857                    }
1858                    TypeBinding::Unbound(_, type_var_kind) => type_var_kind.clone(),
1859                };
1860
1861                // Default any remaining unbound type variables.
1862                // This should only happen if the variable in question is unused
1863                // and within a larger generic type.
1864                let default = match type_var_kind.default_type() {
1865                    Some(typ) => typ,
1866                    None => {
1867                        return Err(MonomorphizationError::NoDefaultType { location });
1868                    }
1869                };
1870
1871                Self::check_type(&default, location)
1872            }
1873
1874            HirType::DataType(_def, args) => {
1875                for arg in args {
1876                    Self::check_type(arg, location)?;
1877                }
1878
1879                Ok(())
1880            }
1881
1882            HirType::Alias(_def, args) => {
1883                for arg in args {
1884                    Self::check_type(arg, location)?;
1885                }
1886
1887                Ok(())
1888            }
1889
1890            HirType::Tuple(fields) => {
1891                for field in fields {
1892                    Self::check_type(field, location)?;
1893                }
1894
1895                Ok(())
1896            }
1897
1898            HirType::Function(args, ret, env, _) => {
1899                for arg in args {
1900                    Self::check_type(arg, location)?;
1901                }
1902
1903                Self::check_type(ret, location)?;
1904                Self::check_type(env, location)
1905            }
1906
1907            HirType::Reference(element, _mutable) => Self::check_type(element, location),
1908            HirType::InfixExpr(lhs, _, rhs, _) => {
1909                Self::check_type(lhs, location)?;
1910                Self::check_type(rhs, location)
1911            }
1912        }
1913    }

1915    /// Check that the 'from' and to' sides of a `CheckedCast` unify and
1916    /// that if the 'to' side evaluates to a field element, then the 'from' side
1917    /// evaluates to the same field element as well.
1918    fn check_checked_cast(
1919        from: &Type,
1920        to: &Type,
1921        location: Location,
1922    ) -> Result<(), MonomorphizationError> {
1923        if from.unify(to).is_err() {
1924            return Err(MonomorphizationError::CheckedCastFailed {
1925                actual: to.clone(),
1926                expected: from.clone(),
1927                location,
1928            });
1929        }
1930        let to_value = to.evaluate_to_signed_field(&to.kind(), location);
1931        if let Ok(to_value) = to_value {
1932            let skip_simplifications = false;
1933            let from_value =
1934                from.evaluate_to_signed_field_helper(&to.kind(), location, skip_simplifications);
1935            if from_value.is_err() || from_value.unwrap() != to_value {
1936                return Err(MonomorphizationError::CheckedCastFailed {
1937                    actual: HirType::Constant(to_value, to.kind()),
1938                    expected: from.clone(),
1939                    location,
1940                });
1941            }
1942        }
1943        Ok(())
1944    }

1946    /// Returns true if a given function type is a closure.
1947    /// Note that this expects the argument to already be a function so that it can
1948    /// distinguish function values (pairs of (constrained, unconstrained)) from closures
1949    /// (function values where each element is itself a tuple), from normal tuples
1950    /// containing functions.
1951    fn is_closure_type(&self, t: &ast::Type) -> bool {

1953            matches!(env.as_ref(), ast::Type::Tuple(_captures))
1954        // All functions should be a pair of (constrained, unconstrained), with
1955        // closures possibly represented in both. So we unpack the tuple and see
1956        // if `constrained` is also a tuple.
1957        } else if let ast::Type::Tuple(elements) = t {
1958            if elements.len() == 2 {
1959                match &elements[0] {
1960                    ast::Type::Tuple(inner_elements) if inner_elements.len() == 2 => {
1961                        matches!(inner_elements[1], ast::Type::Function(..))
1962                    }
1963                    _ => false,
1964                }
1965            } else {
1966                false
1967            }
1968        } else {
1969            false

2037    /// Call the same function with `self.in_unconstrained_fn = true` and `self.in_unconstrained_fn = false`
2038    /// and return a pair of both results. This is most commonly used for functions which are
2039    /// represented as a pair of their constrained and unconstrained versions, with the appropriate
2040    /// version being selected later only once the function is called.
2041    ///
2042    /// If `use_current_runtime` is true we only run the function once with the current runtime.
2043    fn monomorphize_constrained_and_unconstrained<F>(
2044        &mut self,
2045        use_current_runtime: bool,
2046        force_unconstrained: bool,
2047        f: F,
2048    ) -> Result<ast::Expression, MonomorphizationError>
2049    where
2050        F: Clone + FnOnce(&mut Self) -> Result<ast::Expression, MonomorphizationError>,
2051    {
2052        if use_current_runtime {
2053            f(self)
2054        } else {
2055            // Corner case: if force_unconstrained is set the function representation is actually
2056            // (unconstrained, unconstrained) to preserve checks in SSA that all functions are
2057            // unconstrained. This is chosen over the simpler `unconstrained` (no tuple) to keep
2058            // monomorphization consistent regardless of whether the flag is set, since the type
2059            // of functions would also need to be updated, which affects how closures are detected,
2060            // etc.
2061            // If we are in an unconstrained function, then we know that any function we call will
2062            // be monomorphized as unconstrained as well, and we'll never call a constrained one;
2063            // therefore we can never make use of a constrained variant of a lambda, and by not
2064            // generating it we can avoid some illegal corner cases, should the constrained lambda
2065            // that never gets used try to call unconstrained code in its body.
2066            let is_unconstrained = force_unconstrained || self.in_unconstrained_function;
2067
2068            let old_value =
2069                std::mem::replace(&mut self.in_unconstrained_function, is_unconstrained);
2070            let constrained = f.clone()(self)?;
2071
2072            self.in_unconstrained_function = true;
2073            let unconstrained = f(self)?;
2074
2075            self.in_unconstrained_function = old_value;
2076            Ok(ast::Expression::Tuple(vec![constrained, unconstrained]))
2077        }
2078    }

2080    /// Monomorphize a function call.
2081    fn function_call(
2082        &mut self,
2083        call: HirCallExpression,
2084        id: ExprId,
2085    ) -> Result<ast::Expression, MonomorphizationError> {
2086        let original_func = Box::new(self.extract_function(call.func)?);
2087
2088        let crossing_runtime_boundaries =
2089            !self.in_unconstrained_function && self.function_is_unconstrained(call.func);
2090        let is_oracle = self.function_is_oracle(call.func);
2091
2092        if crossing_runtime_boundaries {
2093            self.check_arguments_crossing_runtime_boundaries(&call)?;
2094        }
2095
2096        let func_type = self.interner.id_type(call.func).follow_bindings();
2097        let mut arguments = Vec::with_capacity(call.arguments.len());
2098        if let Type::Function(params, _, _, callee_unconstrained) = &func_type {
2099            assert_eq!(params.len(), call.arguments.len(), "ICE: Unexpected number of call args");
2100            for (typ, id) in params.iter().zip(&call.arguments) {
2101                // If we know that the function expects an unconstrained lambda, or can only make use
2102                // of the unconstrained variant, being itself unconstrained, we can generate a pair of
2103                // two unconstrained functions, and avoid potential illegal passing of mutable references
2104                // from constrained to unconstrained code in a lambda that we know will never be called.
2105                let expr = match typ {
2106                    Type::Function(_, _, _, lambda_unconstrained)
2107                        if *lambda_unconstrained || *callee_unconstrained =>
2108                    {
2109                        let old_force_unconstrained =
2110                            std::mem::replace(&mut self.force_unconstrained, true);
2111                        let expr = self.expr(*id)?;
2112                        self.force_unconstrained = old_force_unconstrained;
2113                        expr
2114                    }
2115                    _ => self.expr(*id)?,
2116                };
2117                arguments.push(expr);
2118            }
2119        } else {
2120            for id in &call.arguments {
2121                arguments.push(self.expr(*id)?);
2122            }
2123        }
2124
2125        let hir_arguments = vecmap(&call.arguments, |id| self.interner.expression(id));
2126
2127        self.patch_debug_instrumentation_call(&call, &original_func, &mut arguments)?;
2128
2129        let return_type = self.interner.id_type(id);
2130        let location = self.interner.expr_location(&id);
2131
2132        if crossing_runtime_boundaries {
2133            self.check_return_type_crossing_runtime_boundaries(&return_type, location)?;
2134        }
2135
2136        if is_oracle {
2137            self.check_return_type_returned_from_oracle(&return_type, location)?;
2138        }
2139
2140        let return_type = Self::convert_type(&return_type, location)?;
2141
2142        let location = call.location;
2143        let func_type = Self::convert_type(&func_type, location)?;
2144        let is_closure = self.is_closure_type(&func_type);
2145
2146        if let ast::Expression::Ident(ident) = original_func.as_ref() {
2147            if let Definition::Oracle(name) = &ident.definition
2148                && let Some(ForeignCall::Print) = ForeignCall::lookup(name)
2149            {
2150                // Oracle calls are required to be wrapped in an unconstrained function
2151                // The first argument to the `print` oracle is a bool, indicating a newline to be inserted at the end of the input
2152                // The second argument is expected to always be an ident
2153                self.append_printable_type_info(&hir_arguments[1], &mut arguments);
2154            }
2155            if let Definition::Builtin(name) = &ident.definition
2156                && name.as_str() == "static_assert"
2157            {
2158                // static_assert can take any type for the `message` argument.
2159                // Here we append printable type info so we can know how to turn that argument
2160                // into a human-readable string.
2161                let typ = self.interner.id_type(call.arguments[1]);
2162                append_printable_type_info_for_type(typ, &mut arguments);
2163            }
2164        }
2165
2166        let mut block_expressions = vec![];
2167
2168        let func = if is_closure {
2169            let local_id = self.next_local_id();
2170
2171            // store the function in a temporary variable before calling it
2172            // this is needed for example if call.func is of the form `foo()()`
2173            // without this, we would translate it to `foo().1(foo().0)`
2174            let let_stmt = ast::Expression::Let(ast::Let {
2175                id: local_id,
2176                mutable: false,
2177                name: "tmp".to_string(),
2178                expression: Box::new(*original_func),
2179            });
2180            block_expressions.push(let_stmt);
2181
2182            let extracted_func = ast::Expression::Ident(ast::Ident {
2183                location: None,
2184                definition: Definition::Local(local_id),
2185                mutable: false,
2186                name: "tmp".to_string(),
2187                typ: Self::convert_type(&self.interner.id_type(call.func), location)?,
2188                id: self.next_ident_id(),
2189            });
2190
2191            let env_argument =
2192                ast::Expression::ExtractTupleField(Box::new(extracted_func.clone()), 0usize);
2193            arguments.insert(0, env_argument);
2194
2195            Box::new(ast::Expression::ExtractTupleField(Box::new(extracted_func), 1usize))
2196        } else {
2197            original_func.clone()
2198        };
2199
2200        let call = self
2201            .try_evaluate_builtin_call(&func, &id, &call.arguments, &arguments, &return_type)?
2202            .unwrap_or(ast::Expression::Call(ast::Call { func, arguments, return_type, location }));
2203
2204        if !block_expressions.is_empty() {
2205            block_expressions.push(call);
2206            Ok(ast::Expression::Block(block_expressions))
2207        } else {
2208            Ok(call)
2209        }
2210    }

2388                None => ast::LValue::Ident(self.local_ident(&ident, &typ)?.unwrap()),
2389            },
2390            HirLValue::MemberAccess { object, field_index, .. } => {
2391                let field_index = field_index.unwrap();
2392                let object = Box::new(self.lvalue(*object)?);
2393                ast::LValue::MemberAccess { object, field_index }
2394            }
2395            HirLValue::Index { array, index, typ, location } => {
2396                let array = Box::new(self.lvalue(*array)?);
2397                let index = Box::new(self.expr(index)?);
2398                let element_type = Self::convert_type(&typ, location)?;
2399                ast::LValue::Index { array, index, element_type, location }
2400            }
2401            HirLValue::Dereference { lvalue, element_type, location, implicitly_added: _ } => {
2402                let reference = Box::new(self.lvalue(*lvalue)?);
2403                let element_type = Self::convert_type(&element_type, location)?;
2404                ast::LValue::Dereference { reference, element_type }
2405            }
2406            HirLValue::Error { .. } => {
2407                unreachable!("Encountered Error node during monomorphization");
2408            }
2409        };
2410
2411        Ok(value)
2412    }

2414    fn lambda(
2415        &mut self,
2416        lambda: HirLambda,
2417        expr: ExprId,
2418    ) -> Result<ast::Expression, MonomorphizationError> {
2419        // Function values are represented as a tuple of (constrained version, unconstrained version)
2420        if lambda.captures.is_empty() {
2421            self.monomorphize_constrained_and_unconstrained(
2422                false,
2423                self.force_unconstrained || lambda.unconstrained,
2424                |this: &mut Self| this.lambda_no_capture(lambda, expr),

2426        } else {
2427            // For closures with captures, we build a shared environment to avoid
2428            // exponential blowup when closures capture other closures.
2429            // Both variants share the same environment.
2430            self.closure_with_shared_env(lambda, expr)
2431        }
2432    }

2484    /// Monomorphize a closure with captures, creating both constrained and unconstrained
2485    /// variants that share the same environment.
2486    ///
2487    /// Sharing the same environment avoids exponential blowup when closures capture other closures.
2488    fn closure_with_shared_env(
2489        &mut self,
2490        lambda: HirLambda,
2491        expr: ExprId,
2492    ) -> Result<ast::Expression, MonomorphizationError> {
2493        let location = self.interner.expr_location(&expr);
2494        let ret_type = Self::convert_type(&lambda.return_type, location)?;
2495        let lambda_name = "lambda";
2496        let parameter_types =
2497            try_vecmap(&lambda.parameters, |(_, typ)| Self::convert_type(typ, location))?;
2498
2499        // Manually convert to Parameters type so we can reuse the self.parameters method
2500        let parameters: Parameters = vecmap(&lambda.parameters, |(pattern, typ)| {
2501            (pattern.clone(), typ.clone(), Visibility::Private)
2502        })
2503        .into();
2504
2505        let converted_parameters = self.parameters(&parameters)?;
2506
2507        // Build the shared environment - captured closures stay as (constrained, unconstrained) pairs
2508        let env_local_id = self.next_local_id();
2509        let env_name = "env";
2510        let env_tuple =
2511            ast::Expression::Tuple(try_vecmap(&lambda.captures, |capture| {
2512                match capture.transitive_capture_index {
2513                    Some(field_index) => {
2514                        let lambda_ctx = self.lambda_envs_stack.last().expect(
2515                            "Expected to find a parent closure environment, but found none",
2516                        );
2517
2518                        let ident = Box::new(ast::Expression::Ident(lambda_ctx.env_ident.clone()));
2519                        Ok(ast::Expression::ExtractTupleField(ident, field_index))
2520                    }
2521                    None => {
2522                        let typ = self.interner.definition_type(capture.ident.id);
2523                        let ident = self.local_ident(&capture.ident, &typ)?.unwrap();
2524                        Ok(ast::Expression::Ident(ident))
2525                    }
2526                }
2527            })?);
2528
2529        let expr_type = self.interner.id_type(expr);
2530        let env_typ = if let Type::Function(_, _, function_env_type, _) = expr_type {
2531            Self::convert_type(&function_env_type, location)?
2532        } else {
2533            unreachable!("expected a Function type for a Lambda node")
2534        };
2535
2536        let env_let_stmt = ast::Expression::Let(ast::Let {
2537            id: env_local_id,
2538            mutable: false,
2539            name: env_name.to_string(),
2540            expression: Box::new(env_tuple),
2541        });
2542
2543        let env_ident = ast::Ident {
2544            location: Some(location),
2545            mutable: true,
2546            definition: Definition::Local(env_local_id),
2547            name: env_name.to_string(),
2548            typ: env_typ.clone(),
2549            id: self.next_ident_id(),
2550        };
2551
2552        // Push the shared environment context for processing both lambda bodies
2553        self.lambda_envs_stack.push(LambdaContext {
2554            env_ident: env_ident.clone(),
2555            captures: lambda.captures.clone(),
2556        });
2557
2558        // Determine if we should force unconstrained for both variants.
2559        // If we're already in an unconstrained context or force_unconstrained is set,
2560        // both variants will be unconstrained, so we only need to create one function.
2561        let force_both_unconstrained = self.force_unconstrained || lambda.unconstrained;
2562        let both_unconstrained = force_both_unconstrained || self.in_unconstrained_function;
2563
2564        let old_unconstrained = self.in_unconstrained_function;
2565
2566        // Build shared parameters structure
2567        let mut parameters =
2568            vec![(env_local_id, true, env_name.to_string(), env_typ.clone(), Visibility::Private)];
2569        parameters.extend(converted_parameters);
2570
2571        // Create constrained variant (or first unconstrained if both are unconstrained)
2572        self.in_unconstrained_function = both_unconstrained;
2573        let constrained_id = self.next_function_id();
2574        let constrained_body = self.expr(lambda.body)?;
2575        let mut lambda_fn = Function {
2576            id: constrained_id,
2577            name: lambda_name.to_owned(),
2578            parameters: parameters.to_vec(),
2579            body: constrained_body,
2580            return_type: ret_type.clone(),
2581            return_visibility: Visibility::Private,
2582            unconstrained: self.in_unconstrained_function,
2583            inline_type: InlineType::default(),
2584            is_entry_point: false,
2585        };
2586        self.push_function(constrained_id, lambda_fn.clone());
2587
2588        // Create unconstrained variant unless the previous variant is already unconstrained
2589        let unconstrained_id = if both_unconstrained {
2590            // Both variants are unconstrained, reuse the same function
2591            constrained_id
2592        } else {
2593            // Create a separate unconstrained variant
2594            self.in_unconstrained_function = true;
2595            let unconstrained_id = self.next_function_id();
2596            let unconstrained_body = self.expr(lambda.body)?;
2597            lambda_fn.id = unconstrained_id;
2598            lambda_fn.unconstrained = true;
2599            lambda_fn.body = unconstrained_body;
2600            self.push_function(unconstrained_id, lambda_fn);
2601            unconstrained_id
2602        };
2603
2604        // Restore state
2605        self.in_unconstrained_function = old_unconstrained;
2606        self.lambda_envs_stack.pop();
2607
2608        // Build the function type for both variants
2609        let constrained_fn_typ = ast::Type::Function(
2610            parameter_types.clone(),
2611            Box::new(ret_type.clone()),
2612            Box::new(env_typ.clone()),
2613            both_unconstrained,
2614        );
2615        let unconstrained_fn_typ = ast::Type::Function(
2616            parameter_types,
2617            Box::new(ret_type),
2618            Box::new(env_typ.clone()),
2619            true,
2620        );
2621
2622        // Build closure tuples: (env, fn) for each variant
2623        let constrained_closure_typ =
2624            ast::Type::Tuple(vec![env_typ.clone(), constrained_fn_typ.clone()]);
2625        let unconstrained_closure_typ =
2626            ast::Type::Tuple(vec![env_typ, unconstrained_fn_typ.clone()]);
2627
2628        // Create the expression that builds both closure variants sharing the same env:
2629        // ((env, constrained_fn), (env, unconstrained_fn))
2630        let constrained_fn_ident = ast::Expression::Ident(ast::Ident {
2631            definition: Definition::Function(constrained_id),
2632            mutable: false,
2633            location: Some(location),
2634            name: lambda_name.to_owned(),
2635            typ: constrained_fn_typ,
2636            id: self.next_ident_id(),
2637        });
2638
2639        let unconstrained_fn_ident = ast::Expression::Ident(ast::Ident {
2640            definition: Definition::Function(unconstrained_id),
2641            mutable: false,
2642            location: Some(location),
2643            name: lambda_name.to_owned(),
2644            typ: unconstrained_fn_typ,
2645            id: self.next_ident_id(),
2646        });
2647
2648        let env_expr = ast::Expression::Ident(env_ident);
2649
2650        let constrained_closure =
2651            ast::Expression::Tuple(vec![env_expr.clone(), constrained_fn_ident]);
2652        let unconstrained_closure = ast::Expression::Tuple(vec![env_expr, unconstrained_fn_ident]);
2653
2654        let closure_pair = ast::Expression::Tuple(vec![constrained_closure, unconstrained_closure]);
2655
2656        // Wrap in a block with the env let statement
2657        let block_local_id = self.next_local_id();
2658        let block_ident_name = "closure_variable";
2659
2660        let block_let_stmt = ast::Expression::Let(ast::Let {
2661            id: block_local_id,
2662            mutable: false,
2663            name: block_ident_name.to_string(),
2664            expression: Box::new(ast::Expression::Block(vec![env_let_stmt, closure_pair])),
2665        });
2666
2667        let result_typ = ast::Type::Tuple(vec![constrained_closure_typ, unconstrained_closure_typ]);
2668
2669        let closure_ident = ast::Expression::Ident(ast::Ident {
2670            location: Some(location),
2671            mutable: false,
2672            definition: Definition::Local(block_local_id),
2673            name: block_ident_name.to_string(),
2674            typ: result_typ,
2675            id: self.next_ident_id(),
2676        });
2677
2678        Ok(ast::Expression::Block(vec![block_let_stmt, closure_ident]))
2679    }

2681    fn match_expr(
2682        &mut self,
2683        match_expr: HirMatch,
2684        expr_id: ExprId,
2685    ) -> Result<ast::Expression, MonomorphizationError> {
2686        let result_type = self.interner.id_type(expr_id);
2687        let location = self.interner.expr_location(&expr_id);
2688
2689        if !self.in_unconstrained_function && result_type.contains_reference() {
2690            let typ = result_type.to_string();
2691            return Err(MonomorphizationError::ReferenceReturnedFromIfOrMatch { typ, location });
2692        }
2693
2694        match match_expr {
2695            HirMatch::Success(id) => self.expr(id),
2696            HirMatch::Failure { .. } => {
2697                let false_ = Box::new(ast::Expression::Literal(ast::Literal::Bool(false)));
2698                let msg = "match failure";
2699                let msg_expr = ast::Expression::Literal(ast::Literal::Str(msg.to_string()));
2700
2701                let length: u32 = msg.len().try_into().expect(
2702                    "ICE: Monomorphizer::match_expr: msg.len() is expected to fit into a u32",
2703                );
2704                let length = length.into();
2705                let msg_type = HirType::String(Box::new(length));
2706
2707                let msg = Some(Box::new((msg_expr, msg_type)));
2708                Ok(ast::Expression::Constrain(false_, location, msg))
2709            }
2710            HirMatch::Guard { cond, body, otherwise } => {
2711                let condition = Box::new(self.expr(cond)?);
2712                let consequence = Box::new(self.expr(body)?);
2713                let alternative = Some(Box::new(self.match_expr(*otherwise, expr_id)?));
2714                let typ = Self::convert_type(&result_type, location)?;
2715                Ok(ast::Expression::If(ast::If { condition, consequence, alternative, typ }))
2716            }
2717            HirMatch::Switch(variable_to_match, cases, default) => {
2718                let variable_name = self.interner.definition(variable_to_match).name.clone();
2719                let variable_id = match self.lookup_local(variable_to_match) {
2720                    Some(Definition::Local(id)) => id,
2721                    other => unreachable!("Expected match variable to be defined. Found {other:?}"),
2722                };
2723
2724                let cases = try_vecmap(cases, |case| {
2725                    let arguments = vecmap(case.arguments, |arg| {
2726                        let arg_name = self.interner.definition(arg).name.clone();
2727                        let new_id = self.next_local_id();
2728                        self.define_local(arg, new_id);
2729                        (new_id, arg_name)
2730                    });
2731                    let branch = self.match_expr(case.body, expr_id)?;
2732                    Ok(ast::MatchCase { constructor: case.constructor, arguments, branch })
2733                })?;
2734
2735                let default_case = match default {
2736                    Some(case) => Some(Box::new(self.match_expr(*case, expr_id)?)),
2737                    None => None,
2738                };
2739
2740                let typ = Self::convert_type(&result_type, location)?;
2741                Ok(ast::Expression::Match(ast::Match {
2742                    variable_to_match: (variable_id, variable_name),
2743                    cases,
2744                    default_case,
2745                    typ,
2746                }))
2747            }
2748        }
2749    }

2751    /// Call an infix operator overloading method for the given operator.
2752    /// This function handles the special cases some operators have which don't map
2753    /// 1 to 1 onto their operator function. For example: != requires a negation on
2754    /// the result of its `eq` method, and the comparison operators each require a
2755    /// conversion from the `Ordering` result to a boolean.
2756    fn create_infix_operator_impl_call(
2757        &self,
2758        func: ast::Expression,
2759        lhs: ast::Expression,
2760        operator: HirBinaryOp,
2761        rhs: ast::Expression,
2762        ret: Type,
2763        location: Location,
2764    ) -> Result<ast::Expression, MonomorphizationError> {
2765        let arguments = vec![lhs, rhs];
2766        let func = Box::new(func);
2767        let return_type = Self::convert_type(&ret, location)?;
2768
2769        let mut result =
2770            ast::Expression::Call(ast::Call { func, arguments, return_type, location });
2771
2772        use crate::ast::BinaryOpKind::*;
2773        match operator.kind {
2774            // Negate the result of the == operation
2775            NotEqual => {
2776                result = ast::Expression::Unary(ast::Unary {
2777                    operator: UnaryOp::Not,
2778                    rhs: Box::new(result),
2779                    result_type: ast::Type::Bool,
2780                    location,
2781                    skip: false,
2782                });
2783            }
2784            // All the comparison operators require special handling since their `cmp` method
2785            // returns an `Ordering` rather than a boolean value.
2786            //
2787            // (a < b) => a.cmp(b) == Ordering::Less
2788            // (a <= b) => a.cmp(b) != Ordering::Greater
2789            // (a > b) => a.cmp(b) == Ordering::Greater
2790            // (a >= b) => a.cmp(b) != Ordering::Less
2791            Less | LessEqual | Greater | GreaterEqual => {
2792                // Comparing an Ordering directly to a field value in this way takes advantage
2793                // of the fact the Ordering struct contains a single Field type, and our SSA
2794                // pass will automatically unpack tuple values.
2795                let ordering_value = if matches!(operator.kind, Less | GreaterEqual) {
2796                    FieldElement::zero() // Ordering::Less
2797                } else {
2798                    2u128.into() // Ordering::Greater
2799                };
2800
2801                let operator =
2802                    if matches!(operator.kind, Less | Greater) { Equal } else { NotEqual };
2803
2804                let ordering_value = SignedField::positive(ordering_value);
2805                let int_value = ast::Literal::Integer(ordering_value, ast::Type::Field, location);
2806                let rhs = Box::new(ast::Expression::Literal(int_value));
2807                let lhs = Box::new(ast::Expression::ExtractTupleField(Box::new(result), 0));
2808
2809                result = ast::Expression::Binary(ast::Binary { lhs, operator, rhs, location });
2810            }
2811            _ => (),
2812        }
2813
2814        Ok(result)
2815    }

2838    // Functions are represented as pairs of (constrained, unconstrained) versions of the same
2839    // function. When calling them we select the version that corresponds to the current
2840    // runtime environment. See https://github.com/noir-lang/noir/issues/7289 for the reasoning.
2841    fn extract_function(
2842        &mut self,
2843        function: ExprId,
2844    ) -> Result<ast::Expression, MonomorphizationError> {
2845        // If it is a known function we can compile only the required version
2846        if let HirExpression::Ident(ident, generics) = self.interner.expression(&function) {
2847            // Check if this directly refers to a function
2848            if matches!(self.interner.definition(ident.id).kind, DefinitionKind::Function(_)) {
2849                return self.ident(ident, function, generics, true, false);
2850            }
2851        }
2852
2853        // Otherwise we fallback to compiling both versions of the function and extracting the
2854        // required one.
2855        let function = self.expr(function)?;
2856        let index = if self.in_unconstrained_function { 1 } else { 0 };
2857
2858        // If this is a tuple literal we can simplify directly. This lets us directly see
2859        // what function we're calling in some cases.
2860        match function {
2861            ast::Expression::Tuple(mut elements) => {
2862                assert_eq!(elements.len(), 2);
2863                Ok(elements.remove(index))
2864            }
2865            tuple => Ok(ast::Expression::ExtractTupleField(Box::new(tuple), index)),
2866        }
2867    }

2870    /// Resolves the callee's type (following bindings) and checks for unconstrained status
2871    /// both on direct function types and on `(constrained_fn, unconstrained_fn)` tuple pairs

2879            }
2880            _ => false,
2881        }
2882    }

3080/// Binds direct generics on a trait impl function to those on a corresponding trait function.
3081///
3082/// Consider this code:
3083///
3084/// ```noir
3085/// trait Bar {
3086///     fn baz<let N: u32>();
3087/// }
3088///
3089/// impl Bar for Field {
3090///     fn baz<let M: u32>() {
3091///         let _ = M;
3092///     }
3093/// }
3094///
3095/// fn foo<K: Bar>() {
3096///     K::baz::<2>();
3097/// }
3098///
3099/// fn main() {
3100///     foo::<Field>();
3101/// }
3102/// ```
3103///
3104/// When type-checking `K::baz::<2>` the compiler will know that it corresponds
3105/// to the method `baz` defined on the `Bar` trait, but won't yet know which
3106/// trait implementation it refers to. The instantiation bindings for this
3107/// will have the type variable `N` bound to `2`.
3108///
3109/// During monomorphization, on the specific call to `foo::<Field>()`,
3110/// the compiler will know that `K` refers to `Field`, and that `K::baz::<2>`
3111/// refers to the `baz` method on the `Bar` impl for `Field`. In that method
3112/// the type variable `M` is unbound. Looking up its value will fail.
3113/// However, we can bind `M` to `N`, so it eventually resolves to `2`,
3114/// which is what this method does.
3115fn bind_trait_impl_func_generics_to_trait_func_generics(
3116    interner: &NodeInterner,
3117    method_id: TraitItemId,
3118    impl_id: node_interner::TraitImplId,
3119    bindings: &mut TypeBindings,
3120) {
3121    // We know the trait method this call resolves to
3122    let trait_func_definition = interner.definition(method_id.item_id);
3123    let DefinitionKind::Function(trait_func_id) = trait_func_definition.kind else {
3124        return;
3125    };
3126    let trait_func_meta = interner.function_meta(&trait_func_id);
3127    let trait_func_name = interner.definition_name(method_id.item_id);
3128
3129    // Find the corresponding trait impl method
3130    let trait_impl = interner.get_trait_implementation(impl_id);
3131    let trait_impl = trait_impl.borrow();
3132    let trait_impl_func_id = trait_impl
3133        .methods
3134        .iter()
3135        .find(|method| interner.function_name(method) == trait_func_name)
3136        .expect("Expected to find trait impl method");
3137
3138    // Bind trait impl func generics to trait func generics
3139    let trait_impl_func_meta = interner.function_meta(trait_impl_func_id);
3140    let trait_impl_func_generics = &trait_impl_func_meta.direct_generics;
3141    let trait_func_generics = &trait_func_meta.direct_generics;
3142    assert_eq!(
3143        trait_impl_func_generics.len(),
3144        trait_func_generics.len(),
3145        "Trait impl function generics and trait function generics should have the same length"
3146    );
3147    for (trait_func_generic, trait_impl_func_generic) in
3148        trait_func_generics.iter().zip(trait_impl_func_generics.iter())
3149    {
3150        let trait_impl_generic = &trait_impl_func_generic.type_var;
3151        bindings.entry(trait_impl_generic.id()).or_insert_with(|| {
3152            (
3153                trait_impl_generic.clone(),
3154                trait_impl_generic.kind(),
3155                Type::TypeVariable(trait_func_generic.type_var.clone()),
3156            )
3157        });
3158    }
3159}

3162/// and associated constants until an item with a matching name is found.
3163///
