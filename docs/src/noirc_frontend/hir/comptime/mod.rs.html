1//! The comptime interpreter is used to execute Noir code which is marked to
2//! be executed at compilation time. The main uses for this are:
3//!     - Metaprogramming such as deriving trait implementations
4//!     - Interpreting comptime blocks, attribute functions, and macros
5//!     - Evaluation of globals
6//!
7//! This interpreter is run on the HIR while the program is being resolved and type-checked, and before it is monomorphized. Code written to be run at comptime may
8//! then require additional type hints compared to equivalent non-comptime Noir code.
9//!
10//! For more information on Noir's comptime execution and metaprogramming in general, see the linked page in
11//! the [Noir docs](<https://noir-lang.org/docs/noir/concepts/comptime>)
12
