Build the tag registry from T::register_into, then deserialize a value of type T from bytes through a Deserializer.

All tagged types are expected to be encoded under Format::MsgpackTagged (int-keyed maps for crate::EncodingStrategy::Tagged, positional arrays for crate::EncodingStrategy::Array — the decoder probes the wire shape per struct). Other formats route through their own decoders upstream via the Format byte (defined in the acir crate).
