## scalar_mul::fixed_base_embedded_curve

Performs scalar multiplication of a fixed base/generator over the embedded curve whose coordinates are defined
by the configured noir field. For the BN254 scalar field, this is BabyJubJub or Grumpkin. Suffixes `_low` and
`_high` denote low and high limbs of the input scalar.

#include_code fixed_base_embedded_curve noir_stdlib/src/scalar_mul.nr rust

example

```rust
fn main(scalar_low: Field, scalar_high: Field) {
    let point = std::scalar_mul::fixed_base_embedded_curve(scalar_low, scalar_high);
    println(point);
}
```

## scalar_mul::variable_base_embedded_curve

Performs scalar multiplication of a variable base/input point over the embedded curve whose coordinates are defined
by the configured noir field. For the BN254 scalar field, this is BabyJubJub or Grumpkin. Suffixes `_low` and
`_high` denote low and high limbs of the input scalar.

#include_code variable_base_embedded_curve noir_stdlib/src/scalar_mul.nr rust

example

```rust
fn main(point_x: Field, point_y: Field, scalar_low: Field, scalar_high: Field) {
    let resulting_point = std::scalar_mul::fixed_base_embedded_curve(point_x, point_y, scalar_low, scalar_high);
    println(resulting_point);
}
```
