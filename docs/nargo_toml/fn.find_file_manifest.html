pub fn find_file_manifest(current_path: &Path) -> Option<PathBuf>
Expand description

Searches for a Nargo.toml file in the current directory and all parent directories. For example, if the current directory is /workspace/package/src, then this function will search for a Nargo.toml file in

  • /workspace/package/src,
  • /workspace/package,
  • /workspace.

Returns the PathBuf of the Nargo.toml file if found, otherwise returns None.

It will return innermost Nargo.toml file, which is the one closest to the current directory. For example, if the current directory is /workspace/package/src, then this function will return the Nargo.toml file in /workspace/package/Nargo.toml
