pub fn add_dependency_to_manifest(
    toml_path: &Path,
    name: &str,
    dep: &DependencyConfig,
    overwrite: bool,
) -> Result<(), ManifestError>
Expand description

Adds a dependency named name to the [dependencies] table of the manifest at toml_path, preserving the file’s existing formatting and comments.

If a dependency with the same name is already present and overwrite is false, this returns ManifestError::DependencyAlreadyExists and leaves the file untouched.
