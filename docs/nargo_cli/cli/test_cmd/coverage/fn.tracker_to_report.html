Expand description

Converts an EvaluationTracker collected during one test run into a Report section keyed by test_name.

Each (test_name, source_file) section is independent and will not be merged with the baseline or other tests when the reports are combined.

Expressions inside the test function’s own body are excluded: they are the test harness, not the code under test.
