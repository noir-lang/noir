Builds the zero-count baseline Report for a compiled package.

Uses an empty test name so its sections (("", file)) are stored under a distinct key from per-test sections (("test_name", file)) and are never merged with them by Report::merge. into_records then emits them as a separate TN: block, giving tools a complete picture of every instrumented line even if no test ever reached it.
