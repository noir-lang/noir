    f: &mut Formatter<'_>,
) -> Result
Expand description

Displays an expression as a quadratic polynomial. If as_equal_to_zero is true, the expression is displayed as equaling zero, where it’s tried to shown as a polynomial equal to the largest witness, if possible. If the optional return_values is provided, the expression is displayed preferring to show ASSERT return_value = ... when possible.
