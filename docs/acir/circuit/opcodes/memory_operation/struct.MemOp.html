Struct MemOp

Source 
pub struct MemOp<F> {
    pub operation: Expression<F>,
    pub index: Expression<F>,
    pub value: Expression<F>,
}
Expand description

Operation on a block of memory We can either write or read at an index in memory

Fields§

§operation: Expression<F>

A constant expression that can be 0 (read) or 1 (write)

§index: Expression<F>

array index, it must be less than the array length

§value: Expression<F>

the value we are reading, when operation is 0, or the value we write at the specified index, when operation is 1
