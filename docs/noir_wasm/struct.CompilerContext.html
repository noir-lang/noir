Struct CompilerContext

Source 
pub struct CompilerContext {
    context: Context<'static, 'static>,
}
Expand description

This is a wrapper class that is wasm-bindgen compatible We do not use js_name and rename it like CrateId because then the impl block is not picked up in javascript.
