Struct DataType

pub struct DataType {
    pub id: TypeId,
    pub name: Ident,
    pub visibility: ItemVisibility,
    body: TypeBody,
    pub generics: ResolvedGenerics,
    pub location: Location,
    pub must_use: MustUse,
}
Expand description

Represents a struct or enum type in the type system. Each instance of this rust struct will be shared across all Type::DataType variants that represent the same struct or enum type.

Fields§

§id: TypeId

A unique id representing this type. Used to check if two types are equal.

§name: Ident§visibility: ItemVisibility§body: TypeBody

A type’s body is private to force struct fields or enum variants to only be accessed through get_field(), get_fields(), instantiate(), or similar functions since these will handle applying generic arguments to fields as well.

§generics: ResolvedGenerics§location: Location§must_use: MustUse

pub fn get_fields_as_written(&self) -> Option<Vec<StructField>>

Returns the name and raw types of each field of this type. This will not substitute any generic arguments so a generic field like x in struct Foo<T> { x: T } will return a ("x", T) pair.

This method is almost never what is wanted for type checking or monomorphization, prefer to use get_fields whenever possible.

Returns None if this is not a struct type.

Source

pub fn get_variants_as_written(&self) -> Option<Vec<EnumVariant>>

Returns the name and raw parameters of each variant of this type. This will not substitute any generic arguments so a generic variant like X in enum Foo<T> { X(T) } will return a ("X", Vec<T>) pair.

Returns None if this is not an enum type.

Source

pub fn get_variant_as_written( &self, variant_index: usize, ) -> Option<&EnumVariant>

Returns the name and raw parameters of the variant at the given variant index. This will not substitute any generic arguments so a generic variant like X in enum Foo<T> { X(T) } will return a ("X", Vec<T>) pair.

Returns None if this is not an enum type or the given variant index is out of bounds.

pub fn variant_function_type( &self, variant_index: usize, this: Shared<DataType>, ) -> Type

Returns the function type of the variant at the given index of this enum. Requires the Shared<DataType> handle of self to create the given function type. Panics if this is not an enum.

The function type uses the variant “as written” ie. no generic substitutions. Although the returned function is technically generic, Type::Function is returned instead of Type::Forall.
