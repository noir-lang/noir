Expand description

Bind the trait’s Self type variable to the impl’s concrete self type when the impl’s matching method shares the trait’s default-method FuncId. In this case the impl has no per-impl FuncId to attach func_id_to_trait to, so function()’s usual force_bind of Self doesn’t run; without this helper the shared body’s references to Self (e.g. Self::N) can’t resolve at monomorphization.

For impl methods with their own FuncId (the typical case), set_function_trait is set during elaboration and function() already binds Self, so this helper is a no-op for them.
