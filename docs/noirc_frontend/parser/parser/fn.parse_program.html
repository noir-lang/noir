Expand description

Entry function for the parser - also handles lexing internally.

Given a source_program string, return the ParsedModule Ast representation of the program along with any parsing errors encountered. If the parsing errors Vec is non-empty, there may be Error nodes in the Ast to fill in the gaps that failed to parse. Otherwise the Ast is guaranteed to have 0 Error nodes.
