pub fn try_to_signed<T>(self) -> Option<T>
where T: TryFrom<u128> + TryFrom<i128> + Signed + Bounded + AbsU128, u128: TryFrom<T>,

Convert a SignedField into a signed integer type (up to i128), returning None if the value does not fit. This function is more complex for handling negative values, specifically INT_MIN which we can’t cast from a u128 to i128 without wrapping it.
