Fields§

§elaborator: &'local mut Elaborator<'interner>

To expand macros the Interpreter needs access to the Elaborator

§crate_id: CrateId§in_loop: bool§current_function: Option<FuncId>§bound_generics: Vec<FxHashMap<TypeVariable, (Type, Kind)>>

Maps each bound generic to each binding it has in the current callstack. Since the interpreter monomorphizes as it interprets, we can bind over the same generic multiple times. Without this map, when one of these inner functions exits we would unbind the generic completely instead of resetting it to its previous binding.

§bigint_solver: BigIntSolverWithId

Stateful bigint calculator.
