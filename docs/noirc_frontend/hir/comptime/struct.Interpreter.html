pub struct Interpreter<'local, 'interner> {
    pub elaborator: &'local mut Elaborator<'interner>,
    in_loop: bool,
    current_function: Option<FuncId>,
    bound_generics: Vec<FxHashMap<TypeVariable, (Type, Kind)>>,
    evaluation_depth: usize,
}

Fields§

§elaborator: &'local mut Elaborator<'interner>

To expand macros the Interpreter needs access to the Elaborator

§in_loop: bool

True if the interpreter is currently in a loop (in the current function). Used only to error if break/continue are used outside a loop.

§current_function: Option<FuncId>

The current function being interpreted. This may be None if we’re interpreting the rhs of a global.

§bound_generics: Vec<FxHashMap<TypeVariable, (Type, Kind)>>

Maps each generic to the binding it has in the current callstack. Since the interpreter monomorphizes as it interprets, we can bind over the same generic multiple times. Without the outer Vec, when one of these inner functions exits we would unbind the generic completely instead of resetting it to its previous binding.

§evaluation_depth: usize

Current evaluation depth.
