fn base_ident_of_field_access(expr: &Expression) -> Option<LocalId>
Expand description

Given an expression that is the operand of a reference (&expr or &mut expr), walk through any chain of struct-field accesses (expr.field = ExtractTupleField) and return the LocalId of the base variable, if it is a local variable.

For example:

  • &mut x → Some(x_id)
  • &mut x.field → Some(x_id) (field is ExtractTupleField(x, _))
  • &mut x.a.b → Some(x_id)
  • &mut some_call() → None
