Function fixpoint

fn fixpoint<S: Default>(
    ssa: &Ssa,
    init: impl FnMut(&Function, &mut S),
    update: impl FnMut(&Function, &mut S) -> bool,
) -> S
Expand description

Populate per-function state with init, then run update over every function until a full round makes no change, and return the final state.

init receives each function and a mutable reference to the default-initialized state to fill in that function’s entry. update likewise receives each function and the state, mutates it in place, and returns whether it changed anything; returning false short-circuits a function whose contribution is already settled. The caller keeps whichever part of the final state it needs.
