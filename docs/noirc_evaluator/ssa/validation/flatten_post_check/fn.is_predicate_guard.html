fn is_predicate_guard(
    dfg: &DataFlowGraph,
    instruction: &Instruction,
    operand: ValueId,
    p: ValueId,
) -> bool
Expand description

True if instruction re-applies predicate p to operand, zeroing it whenever p is false: mul p, operand (either order), or a branch of an IfElse whose condition is p and whose value is operand. A merge gates each branch by its own condition (then_condition * then_value + else_condition * else_value), so both branches are checked.
