    pub current_function: Function,

}
Expand description

The per-function context for each ssa function being generated.

This is split from the global SsaBuilder context to allow each function to be potentially built concurrently.

Contrary to the name, this struct has the capacity to build as many functions as needed, although it is limited to one function at a time.

fn update_array_reference_count( &mut self, value: ValueId, increment: bool, ) -> Option<ValueId>

Increment or decrement the given value’s reference count if it is an array. If it is not an array, this does nothing. Note that inc_rc and dec_rc instructions are ignored outside of unconstrained code.

If there is an original value it indicates that we’re now decrementing it, but that should only happen if it hasn’t been changed by an array_set in the meantime, which in itself would make sure its RC is decremented.

Returns the ID of the array that was affected, if any.
