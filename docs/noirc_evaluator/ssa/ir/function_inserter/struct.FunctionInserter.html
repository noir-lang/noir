pub(crate) struct FunctionInserter<'f> {
    pub(crate) function: &'f mut Function,
    values: FxHashMap<ValueId, ValueId>,
}
Expand description

The FunctionInserter can be used to help modify existing Functions and map old values to new values after re-inserting optimized versions of old instructions.
