Enum Tree

pub(super) enum Tree<T> {
    Branch(Vec<Tree<T>>),
    Leaf(T),
}
Expand description

A general Tree structure which is used in the SSA generation pass to represent both values and types which may be tuples.

Since the underlying SSA intermediate representation (IR) does not support tuples directly, theyโre instead represented as Tree::Branch nodes. For example, a single ssa value may be a Tree::Leaf(Value), while a tuple would be a Tree::Branch(values).

pub(super) fn map<U>(self, f: impl FnMut(T) -> Tree<U>) -> Tree<U>

Calls the given function on each leaf node, mapping this tree into a new one.

Because the given function returns a Tree<U> rather than a U, it is possible to use this function to turn Leaf nodes into either other Leaf nodes or even Branch nodes.
