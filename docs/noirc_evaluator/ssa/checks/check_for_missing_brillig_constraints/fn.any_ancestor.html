fn any_ancestor(
    start: ValueId,
    predicate: impl Fn(ValueId) -> bool,
    parents: &HashMap<ValueId, Vec<ValueId>>,
    equivalences: &HashMap<ValueId, Vec<ValueId>>,
    max_ancestor_distance: u32,
) -> bool
Expand description

Returns true if start itself, or any value reachable from start by following parents (and equivalences from intermediate nodes), satisfies a predicate.

Equivalences are not followed directly from start — only from nodes reached via parent edges. See bfs_ancestors for the rationale.
