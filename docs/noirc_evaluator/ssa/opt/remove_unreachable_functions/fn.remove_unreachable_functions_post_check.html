fn remove_unreachable_functions_post_check(ssa: &Ssa)
Expand description

Post-check condition for Ssa::remove_unreachable_functions.

Succeeds if:

  • ssa contains no unreachable functions, i.e., all functions are reachable from the entry points.

Otherwise panics.

ยงNote

We reuse the same logic for checking reachability as in the main pass, so this function will not catch any bugs in the pass itself, but it will ensure that the pass is idempotent.
