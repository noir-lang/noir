This SSA pass replaces Shl and Shr instructions in ACIR functions with more primitive arithmetic instructions since ACIR doesnâ€™t directly support bit shifts.

In all cases, if the shift amount is equal to or exceeds the operandâ€™s number of bits, the result will be a constrain failure (attempt to bit-shift with overflow).

Â§Unsigned shift-right

Shifting an unsigned integer to the right by N is the same as diving by 2^N:

// this:
v2 = shr v1, 3

// is replaced with:
v2 = div v1, 8

If the shift amount is not a constant, 2^N is computed via square&multiply, using the bits decomposition of exponent.

Pseudo-code of the computation:

let mut r = 1;
let exponent_bits = to_bits(exponent);
for i in 1 .. bit_size + 1 {
    let r_squared = r * r;
    let b = exponent_bits[bit_size - i];
    r = if b { 2 * r_squared } else { r_squared };
}

Shifting an unsigned integer to the right by N is the same as multiplying by 2^N. However, since that can overflow the target bit size, the operation is done using Field, then truncated to the target bit size.

// this, assuming v1 is a u8:
v2 = shl v1, 3

// is replaced with:
v2 = cast v1 as Field
v3 = mul v2, 8
v4 = truncate v3 to 8 bits, max_bit_size: 11
v5 = cast v4 as u8
