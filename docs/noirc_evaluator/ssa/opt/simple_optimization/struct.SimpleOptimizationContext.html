pub(crate) fn insert_current_instruction(&mut self)

Instructs this context to insert the current instruction right away, as opposed to doing this at the end of mutate’s block (unless remove_current_instruction is called).

If the instruction or its values has changed relative to their original content, we attempt to simplify the instruction before re-inserting it into the block.
