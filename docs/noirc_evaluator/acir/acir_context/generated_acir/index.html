Module generated_acir

Source
Expand description

The generated_acir module is responsible for the lowest level tracking of ACIR-gen.

This module defines the GeneratedAcir which is constructed as part of the ACIR-gen pass to accumulate all of the ACIR. This tracks properties such as which witnesses form part of the circuit interface and defines common snippets of ACIR to perform simple operations. Notably it has no concept of SSA or any types for the variables it works with, it instead only works on the Expressions and Witnesses which will be reflected in the final circuit.
