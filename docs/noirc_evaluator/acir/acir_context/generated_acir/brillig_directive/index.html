PLACEHOLDER_BRILLIG_INDEX 🔒
Brillig calls such as for the Brillig std lib are resolved only after code generation is finished. This index should be used when adding a Brillig call during code generation. Code generation should then keep track of that unresolved call opcode which will be resolved with the correct function index after code generation.
