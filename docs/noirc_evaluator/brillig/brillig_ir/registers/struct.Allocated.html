Wrapper for a memory address which automatically deallocates itself when it goes out of scope.

pub(crate) fn detach(self) -> A

Detach the allocated value from the registers, promising to take care of ownership in a different way.

This is used when we allocate an address in a register, then create a new register where we consider the value pre-allocated, and finally want to deallocate the address in the new register, not the old one, so that we can reuse the memory for something else.
