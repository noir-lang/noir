fn is_fallible_black_box_op(op: &BlackBoxOp) -> bool

Side-effecting / failing black-box ops. Delegates to ACIR’s acvm::acir::BlackBoxFunc::has_side_effects for the ACIR-mapped variants (so any future change to that classification flows through here automatically), with BlackBoxOp::ToRadix handled explicitly because the brillig VM validates its radix argument against [2, 256] at runtime and ToRadix has no ACIR analog.
