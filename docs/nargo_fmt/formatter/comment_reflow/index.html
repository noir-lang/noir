Module comment_reflow

Source
Expand description

Paragraph-aware reflow for comments.

Given a sequence of comment content lines (with the per-line prefix such as //, ///, or * already stripped), this module merges adjacent “plain” lines into paragraphs, then greedily wraps each paragraph to the configured width. Certain line shapes (markdown headers, list items, blockquotes, fenced code, tables, URL lines, Javadoc-style @tag lines, blank lines) are recognized and never merged into prose.

Output lines do not carry the prefix or any caller-side indentation; the caller is responsible for adding them. The engine is a pure function with no formatter state.
